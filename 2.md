Да. Ваша текущая проблема в том, что для SQL к LiveBook-таблицам вы пытаетесь использовать **глобальный индекс всех полей всех классов**. Для LiveBook это неправильно.

Для SQL к таблице LiveBook нужно использовать **схему конкретного LiveBook**, которая строится из запроса создания LiveBook:

```json
{
  "liveBookId": 250,
  "liveBookName": "LiveBook250",
  "fields": [
    "Entity.id",
    "FxSpotForwardTrade.baseAmount",
    "FxSpotForwardTrade.baseCurrency.code",
    "Trade.portfolioId"
  ]
}
```

Из этого должен получиться локальный индекс только для этой таблицы:

```text
fieldKey/alias             physical ClickHouse column      type
--------------------------------------------------------------------
id                         id                              LONG
baseAmount                 baseAmount                      NUMERIC
base_amount                baseAmount                      NUMERIC
baseCurrency_code          baseCurrency_code               STRING
base_currency_code         baseCurrency_code               STRING
portfolioId                portfolioId                     LONG
portfolio_id               portfolioId                     LONG
```

Тогда ошибка типа:

```text
Ambiguous fieldKey 'instrument_id'
```

исчезает, потому что `instrument_id` будет искаться **не во всех классах**, а только в схеме конкретного LiveBook.

# 2. Немного доработайте `BusinessMetadataService`

Сейчас `BusinessMetadataService` хранит только `BusinessMetadataRegistry`.  
Но для резолва путей LiveBook нам нужно иметь исходное дерево классов.

Добавьте хранение `roots`.

```java
package com.pivotService.metadata;

import com.fasterxml.jackson.core.type.TypeReference;
import com.fasterxml.jackson.databind.JsonNode;
import com.fasterxml.jackson.databind.ObjectMapper;
import com.pivotService.metadata.dto.BusinessClassNode;
import org.springframework.stereotype.Service;

import java.util.List;
import java.util.concurrent.atomic.AtomicReference;

@Service
public class BusinessMetadataService {

    private final ObjectMapper objectMapper;

    private final AtomicReference<BusinessMetadataRegistry> registryRef =
            new AtomicReference<>(BusinessMetadataRegistry.empty());

    private final AtomicReference<List<BusinessClassNode>> rootsRef =
            new AtomicReference<>(List.of());

    public BusinessMetadataService(ObjectMapper objectMapper) {
        this.objectMapper = objectMapper;
    }

    public void refresh(JsonNode hierarchyJson) {
        try {
            List<BusinessClassNode> roots;

            if (hierarchyJson.isArray()) {
                roots = objectMapper.convertValue(
                        hierarchyJson,
                        new TypeReference<List<BusinessClassNode>>() {
                        }
                );
            } else {
                roots = List.of(objectMapper.treeToValue(hierarchyJson, BusinessClassNode.class));
            }

            rootsRef.set(List.copyOf(roots));
            registryRef.set(BusinessMetadataRegistry.from(roots));
        } catch (Exception e) {
            throw new IllegalArgumentException("Cannot parse business metadata hierarchy", e);
        }
    }

    public BusinessMetadataRegistry registry() {
        BusinessMetadataRegistry registry = registryRef.get();

        if (registry.isEmpty()) {
            throw new IllegalStateException("Business metadata registry is empty");
        }

        return registry;
    }

    public List<BusinessClassNode> roots() {
        List<BusinessClassNode> roots = rootsRef.get();

        if (roots.isEmpty()) {
            throw new IllegalStateException("Business metadata roots are empty");
        }

        return roots;
    }
}
```

# 5. `LiveBookPathResolver`

Это адаптация вашего `DataDictionaryPathResolver`, но под текущие DTO:

- `BusinessClassNode`;
- `BusinessField`;
- `BusinessRelation`.

Главное отличие: для имени физической колонки используется `jsonName`, а не сырой токен из пути.

```java
package com.pivotService.livebook.schema;

import com.pivotService.metadata.FieldKind;
import com.pivotService.metadata.dto.BusinessClassNode;
import com.pivotService.metadata.dto.BusinessField;
import com.pivotService.metadata.dto.BusinessRelation;

import java.util.*;
import java.util.stream.Collectors;

public class LiveBookPathResolver {

    private record FieldInfo(BusinessField field) {
        String jsonName() {
            return field.effectiveJsonName();
        }

        String type() {
            return field.type();
        }

        FieldKind kind() {
            return FieldKind.from(field.type());
        }

        String clickHouseType() {
            return ClickHouseLiveBookTypeMapper.toClickHouseType(field.type());
        }
    }

    private final Map<String, BusinessClassNode> bySourceValue = new HashMap<>();
    private final Map<BusinessClassNode, BusinessClassNode> parentOf = new IdentityHashMap<>();

    public LiveBookPathResolver(List<BusinessClassNode> roots) {
        for (BusinessClassNode root : roots) {
            index(root, null);
        }
    }

    private void index(BusinessClassNode node, BusinessClassNode parent) {
        if (node == null) {
            return;
        }

        if (hasText(node.sourceValue())) {
            bySourceValue.putIfAbsent(node.sourceValue(), node);
        }

        if (parent != null) {
            parentOf.put(node, parent);
        }

        for (BusinessClassNode child : node.children()) {
            index(child, node);
        }
    }

    public List<ResolvedLiveBookField> resolveAll(List<String> paths) {
        if (paths == null) {
            return List.of();
        }

        return paths.stream()
                .map(this::resolve)
                .toList();
    }

    public ResolvedLiveBookField resolve(String path) {
        if (!hasText(path)) {
            throw new IllegalArgumentException("LiveBook field path is empty");
        }

        String[] tokens = path.split("\\.");

        if (tokens.length < 2) {
            throw new IllegalArgumentException("Expected path Class.field or Class.relation.field: " + path);
        }

        BusinessClassNode current = bySourceValue.get(tokens[0]);

        if (current == null) {
            throw new NoSuchElementException(
                    "No entity with sourceValue='" + tokens[0] + "' for path " + path
            );
        }

        List<List<String>> segments = new ArrayList<>();
        List<String> segment = new ArrayList<>();

        BusinessClassNode wholeSetTarget = null;
        FieldInfo leaf = null;

        for (int i = 1; i < tokens.length; i++) {
            String token = tokens[i];
            boolean last = i == tokens.length - 1;

            if (last) {
                BusinessField field = findFieldInherited(current, token);

                if (field != null) {
                    segment.add(field.effectiveJsonName());
                    segments.add(segment);
                    leaf = new FieldInfo(field);
                    break;
                }

                BusinessRelation embeddedSet = findRelationInherited(current, token);

                if (embeddedSet != null && isEmbeddedSet(embeddedSet)) {
                    segment.add(relationSegment(embeddedSet, token));
                    segments.add(segment);

                    BusinessClassNode target = bySourceValue.get(embeddedSet.targetClass());

                    if (target == null) {
                        throw new NoSuchElementException(
                                "No targetClass='" + embeddedSet.targetClass() + "' for path " + path
                        );
                    }

                    wholeSetTarget = target;
                    break;
                }

                throw new NoSuchElementException(
                        "Field or embedded_set '" + token + "' not found in '" +
                                current.sourceValue() + "' for path " + path
                );
            }

            BusinessRelation relation = findRelationInherited(current, token);

            if (relation == null) {
                throw new NoSuchElementException(
                        "Relation '" + token + "' not found in '" +
                                current.sourceValue() + "' for path " + path
                );
            }

            segment.add(relationSegment(relation, token));

            if (isEmbeddedSet(relation)) {
                segments.add(segment);
                segment = new ArrayList<>();
            }

            BusinessClassNode next = bySourceValue.get(relation.targetClass());

            if (next == null) {
                throw new NoSuchElementException(
                        "No targetClass='" + relation.targetClass() +
                                "' for relation '" + token + "' in path " + path
                );
            }

            current = next;
        }

        if (wholeSetTarget == null) {
            if (leaf == null) {
                throw new IllegalStateException("Cannot resolve leaf field for path " + path);
            }

            if (segments.size() == 1) {
                String key = String.join("_", segments.get(0));

                return new ResolvedLiveBookField(
                        path,
                        key,
                        leaf.field(),
                        leaf.type(),
                        leaf.kind(),
                        leaf.clickHouseType(),
                        segments.get(0),
                        null
                );
            }

            List<String> firstArray = segments.get(0);
            List<List<String>> inner = new ArrayList<>(segments.subList(1, segments.size()));

            String group = String.join("_", firstArray);

            String subName = inner.stream()
                    .flatMap(List::stream)
                    .collect(Collectors.joining("_"));

            ResolvedLiveBookField.NestedColumn nestedColumn =
                    new ResolvedLiveBookField.NestedColumn(
                            subName,
                            leaf.field(),
                            leaf.type(),
                            leaf.kind(),
                            leaf.clickHouseType(),
                            inner
                    );

            return new ResolvedLiveBookField(
                    path,
                    group,
                    null,
                    null,
                    null,
                    null,
                    firstArray,
                    List.of(nestedColumn)
            );
        }

        List<String> firstArray = segments.get(0);
        List<List<String>> midInner = new ArrayList<>(segments.subList(1, segments.size()));

        String group = String.join("_", firstArray);

        List<ResolvedLiveBookField.NestedColumn> nestedColumns = new ArrayList<>();

        for (FieldInfo fieldInfo : collectInheritedFields(wholeSetTarget)) {
            List<List<String>> inner = new ArrayList<>(midInner);
            inner.add(List.of(fieldInfo.jsonName()));

            String subName = inner.stream()
                    .flatMap(List::stream)
                    .collect(Collectors.joining("_"));

            nestedColumns.add(new ResolvedLiveBookField.NestedColumn(
                    subName,
                    fieldInfo.field(),
                    fieldInfo.type(),
                    fieldInfo.kind(),
                    fieldInfo.clickHouseType(),
                    inner
            ));
        }

        if (nestedColumns.isEmpty()) {
            throw new NoSuchElementException(
                    "Target class '" + wholeSetTarget.sourceValue() +
                            "' has no fields for path " + path
            );
        }

        return new ResolvedLiveBookField(
                path,
                group,
                null,
                null,
                null,
                null,
                firstArray,
                nestedColumns
        );
    }

    private BusinessField findFieldInherited(BusinessClassNode node, String token) {
        for (BusinessClassNode current = node; current != null; current = parentOf.get(current)) {
            for (BusinessField field : current.fields()) {
                if (token.equals(field.effectiveJsonName()) || token.equals(field.effectiveName())) {
                    return field;
                }
            }
        }

        return null;
    }

    private List<FieldInfo> collectInheritedFields(BusinessClassNode node) {
        LinkedHashMap<String, BusinessField> fieldsByJsonName = new LinkedHashMap<>();

        for (BusinessClassNode current = node; current != null; current = parentOf.get(current)) {
            for (BusinessField field : current.fields()) {
                fieldsByJsonName.putIfAbsent(field.effectiveJsonName(), field);
            }
        }

        return fieldsByJsonName.values()
                .stream()
                .map(FieldInfo::new)
                .toList();
    }

    private BusinessRelation findRelationInherited(BusinessClassNode node, String token) {
        for (BusinessClassNode current = node; current != null; current = parentOf.get(current)) {
            for (BusinessRelation relation : current.relations()) {
                if (relationMatches(relation, token)) {
                    return relation;
                }
            }
        }

        return null;
    }

    private boolean relationMatches(BusinessRelation relation, String token) {
        if (isEmbeddedSet(relation)) {
            return token.equals(relation.name()) || token.equals(relation.alias());
        }

        return token.equals(relation.alias()) || token.equals(relation.name());
    }

    private String relationSegment(BusinessRelation relation, String fallbackToken) {
        if (isEmbeddedSet(relation)) {
            if (hasText(relation.name())) {
                return relation.name();
            }

            if (hasText(relation.alias())) {
                return relation.alias();
            }

            return fallbackToken;
        }

        if (hasText(relation.alias())) {
            return relation.alias();
        }

        if (hasText(relation.name())) {
            return relation.name();
        }

        return fallbackToken;
    }

    private boolean isEmbeddedSet(BusinessRelation relation) {
        return "embedded_set".equalsIgnoreCase(relation.type());
    }

    private boolean hasText(String value) {
        return value != null && !value.isBlank();
    }
}
```

# 7. `LiveBookSchema`

Это локальная схема конкретного LiveBook.

```java
package com.pivotService.livebook.schema;

import java.util.*;

public final class LiveBookSchema {

    private final Long liveBookId;
    private final String liveBookName;
    private final String tableName;
    private final List<LiveBookColumn> columns;
    private final Map<String, LiveBookColumn> columnsByKey;

    private LiveBookSchema(
            Long liveBookId,
            String liveBookName,
            String tableName,
            List<LiveBookColumn> columns,
            Map<String, LiveBookColumn> columnsByKey
    ) {
        this.liveBookId = liveBookId;
        this.liveBookName = liveBookName;
        this.tableName = tableName;
        this.columns = List.copyOf(columns);
        this.columnsByKey = Map.copyOf(columnsByKey);
    }

    public static LiveBookSchema fromResolved(
            Long liveBookId,
            String liveBookName,
            String tableName,
            List<ResolvedLiveBookField> resolvedFields
    ) {
        List<LiveBookColumn> columns = flatten(resolvedFields);
        Map<String, LiveBookColumn> index = buildIndex(columns);

        return new LiveBookSchema(
                liveBookId,
                liveBookName,
                tableName,
                columns,
                index
        );
    }

    public Long liveBookId() {
        return liveBookId;
    }

    public String liveBookName() {
        return liveBookName;
    }

    public String tableName() {
        return tableName;
    }

    public List<LiveBookColumn> columns() {
        return columns;
    }

    public Optional<LiveBookColumn> findColumn(String fieldKey) {
        if (fieldKey == null || fieldKey.isBlank()) {
            return Optional.empty();
        }

        return Optional.ofNullable(columnsByKey.get(fieldKey));
    }

    public LiveBookColumn requireColumn(String fieldKey) {
        return findColumn(fieldKey)
                .orElseThrow(() -> new IllegalArgumentException(
                        "Unknown fieldKey '" + fieldKey +
                                "' in LiveBook schema '" + liveBookName + "'"
                ));
    }

    private static List<LiveBookColumn> flatten(List<ResolvedLiveBookField> resolvedFields) {
        List<LiveBookColumn> result = new ArrayList<>();

        for (ResolvedLiveBookField resolved : resolvedFields) {
            if (!resolved.isNested()) {
                String ddlType = "id".equals(resolved.key())
                        ? resolved.clickHouseType()
                        : "Nullable(" + resolved.clickHouseType() + ")";

                result.add(new LiveBookColumn(
                        resolved.originalPath(),
                        resolved.key(),
                        ddlType,
                        resolved.type(),
                        resolved.kind(),
                        resolved.clickHouseType(),
                        resolved.field(),
                        false
                ));

                continue;
            }

            for (ResolvedLiveBookField.NestedColumn nested : resolved.nested()) {
                String columnName = resolved.key() + "." + nested.name();

                String ddlType = "Nullable(" + nested.clickHouseType() + ")";

                for (int depth = 0; depth < nested.segments().size(); depth++) {
                    ddlType = "Array(" + ddlType + ")";
                }

                result.add(new LiveBookColumn(
                        resolved.originalPath(),
                        columnName,
                        ddlType,
                        nested.type(),
                        nested.kind(),
                        nested.clickHouseType(),
                        nested.field(),
                        true
                ));
            }
        }

        return result;
    }

    private static Map<String, LiveBookColumn> buildIndex(List<LiveBookColumn> columns) {
        Map<String, LiveBookColumn> result = new LinkedHashMap<>();

        for (LiveBookColumn column : columns) {
            registerAlias(result, column.columnName(), column);
            registerAlias(result, toSnakeCase(column.columnName()), column);
            registerAlias(result, column.columnName().replace(".", "_"), column);
            registerAlias(result, toSnakeCase(column.columnName().replace(".", "_")), column);
            registerAlias(result, column.originalPath(), column);

            /*
             * Регистрируем field.name и field.jsonName только для простых скалярных полей.
             *
             * Например:
             *   physical column: baseAmount
             *   field.name:      base_amount
             *
             * Но для FxSpotForwardTrade.baseCurrency.code physical column будет baseCurrency_code.
             * Там НЕ надо регистрировать alias "code", потому что таких code может быть много.
             */
            if (isSimpleScalarColumn(column)) {
                registerAlias(result, column.sourceField().effectiveName(), column);
                registerAlias(result, column.sourceField().effectiveJsonName(), column);
            }
        }

        return result;
    }

    private static boolean isSimpleScalarColumn(LiveBookColumn column) {
        return !column.nested()
                && column.sourceField() != null
                && column.columnName().equals(column.sourceField().effectiveJsonName());
    }

    private static void registerAlias(
            Map<String, LiveBookColumn> index,
            String alias,
            LiveBookColumn column
    ) {
        if (alias == null || alias.isBlank()) {
            return;
        }

        LiveBookColumn existing = index.get(alias);

        if (existing == null) {
            index.put(alias, column);
            return;
        }

        if (samePhysicalColumn(existing, column)) {
            return;
        }

        throw new IllegalArgumentException(
                "Ambiguous LiveBook field alias '" + alias + "'. " +
                        "Column '" + existing.columnName() + "' conflicts with column '" +
                        column.columnName() + "'"
        );
    }

    private static boolean samePhysicalColumn(LiveBookColumn left, LiveBookColumn right) {
        return Objects.equals(left.columnName(), right.columnName())
                && left.kind() == right.kind();
    }

    private static String toSnakeCase(String value) {
        if (value == null || value.isBlank()) {
            return value;
        }

        String normalized = value.replace(".", "_");

        return normalized
                .replaceAll("([a-z0-9])([A-Z])", "$1_$2")
                .replaceAll("([A-Z]+)([A-Z][a-z])", "$1_$2")
                .toLowerCase(Locale.ROOT);
    }
}
```

# 8. `LiveBookSchemaBuilder`

```java
package com.pivotService.livebook.schema;

import com.pivotService.metadata.BusinessMetadataService;
import org.springframework.stereotype.Service;

import java.util.List;
import java.util.Locale;

@Service
public class LiveBookSchemaBuilder {

    private final BusinessMetadataService metadataService;

    public LiveBookSchemaBuilder(BusinessMetadataService metadataService) {
        this.metadataService = metadataService;
    }

    public LiveBookSchema build(
            Long liveBookId,
            String liveBookName,
            List<String> fieldPaths
    ) {
        if (liveBookName == null || liveBookName.isBlank()) {
            throw new IllegalArgumentException("liveBookName is required");
        }

        if (fieldPaths == null || fieldPaths.isEmpty()) {
            throw new IllegalArgumentException("LiveBook fields are required");
        }

        LiveBookPathResolver resolver =
                new LiveBookPathResolver(metadataService.roots());

        List<ResolvedLiveBookField> resolvedFields =
                resolver.resolveAll(fieldPaths);

        return LiveBookSchema.fromResolved(
                liveBookId,
                liveBookName,
                normalizeTableName(liveBookName),
                resolvedFields
        );
    }

    private String normalizeTableName(String liveBookName) {
        /*
         * У вас пример:
         *   liveBookName: LiveBook250
         *   table:        livebook250
         *
         * Если в вашем ClickHouse реально таблицы создаются lower-case,
         * то оставьте toLowerCase().
         */
        return liveBookName.toLowerCase(Locale.ROOT);
    }
}
```

# 9. `LiveBookSchemaRegistry`

Пока можно сделать in-memory registry.

```java
package com.pivotService.livebook.schema;

import org.springframework.stereotype.Service;

import java.util.Locale;
import java.util.Map;
import java.util.Optional;
import java.util.concurrent.ConcurrentHashMap;

@Service
public class LiveBookSchemaRegistry {

    private final Map<String, LiveBookSchema> byName = new ConcurrentHashMap<>();
    private final Map<Long, LiveBookSchema> byId = new ConcurrentHashMap<>();

    public void register(LiveBookSchema schema) {
        if (schema == null) {
            return;
        }

        putName(schema.liveBookName(), schema);
        putName(schema.tableName(), schema);

        if (schema.liveBookId() != null) {
            byId.put(schema.liveBookId(), schema);
        }
    }

    public Optional<LiveBookSchema> findByName(String liveBookNameOrTableName) {
        if (liveBookNameOrTableName == null || liveBookNameOrTableName.isBlank()) {
            return Optional.empty();
        }

        return Optional.ofNullable(byName.get(normalize(liveBookNameOrTableName)));
    }

    public Optional<LiveBookSchema> findById(Long liveBookId) {
        if (liveBookId == null) {
            return Optional.empty();
        }

        return Optional.ofNullable(byId.get(liveBookId));
    }

    public LiveBookSchema requireByName(String liveBookNameOrTableName) {
        return findByName(liveBookNameOrTableName)
                .orElseThrow(() -> new IllegalArgumentException(
                        "LiveBook schema not found: " + liveBookNameOrTableName
                ));
    }

    private void putName(String name, LiveBookSchema schema) {
        if (name == null || name.isBlank()) {
            return;
        }

        byName.put(normalize(name), schema);
    }

    private String normalize(String value) {
        return value.toLowerCase(Locale.ROOT);
    }
}
```

> Важно: in-memory registry потеряется после рестарта приложения.  
> В production лучше сохранять `LiveBookSchema` в БД или заново получать схемы LiveBook-ов из сервиса, который их создает.

# 10. `ClickHouseLiveBookDdlBuilder`

Если вам нужно генерировать `CREATE TABLE` в этом сервисе:

```java
package com.pivotService.livebook.schema;

import org.springframework.stereotype.Component;

import java.util.StringJoiner;

@Component
public class ClickHouseLiveBookDdlBuilder {

    public String buildCreateTable(LiveBookSchema schema) {
        StringJoiner columns = new StringJoiner(",\n    ");

        for (LiveBookColumn column : schema.columns()) {
            columns.add(quoteColumn(column.columnName()) + " " + column.ddlType());
        }

        columns.add(quoteColumn("_version") + " DateTime64(3) DEFAULT now64(3)");

        return """
                CREATE TABLE IF NOT EXISTS %s
                (
                    %s
                )
                ENGINE = ReplacingMergeTree(_version)
                ORDER BY %s
                SETTINGS index_granularity = 8192
                """.formatted(
                quoteTable(schema.tableName()),
                columns,
                quoteColumn("id")
        );
    }

    /**
     * Важно: columnName может содержать точку, например:
     *   alternativeIdentifiers.sourceId
     *
     * Поэтому колонку нужно квотить целиком:
     *   "alternativeIdentifiers.sourceId"
     *
     * А НЕ так:
     *   "alternativeIdentifiers"."sourceId"
     */
    public String quoteColumn(String columnName) {
        if (columnName == null || columnName.isBlank()) {
            throw new IllegalArgumentException("Column name is empty");
        }

        return "\"" + columnName.replace("\"", "\"\"") + "\"";
    }

    /**
     * Для tableName точка означает database.table.
     */
    public String quoteTable(String tableName) {
        if (tableName == null || tableName.isBlank()) {
            throw new IllegalArgumentException("Table name is empty");
        }

        String[] parts = tableName.split("\\.");

        StringJoiner joiner = new StringJoiner(".");

        for (String part : parts) {
            joiner.add("\"" + part.replace("\"", "\"\"") + "\"");
        }

        return joiner.toString();
    }
}
```

Для вашего примера схема даст колонки:

```sql
"id" Int64,
"baseAmount" Nullable(Decimal(18, 6)),
"baseCurrency_code" Nullable(String),
"portfolioId" Nullable(Int64),
"_version" DateTime64(3) DEFAULT now64(3)
```

# 11. Как использовать при создании LiveBook

Допустим, у вас есть request:

```java
package com.pivotService.livebook;

import java.util.List;

        Long liveBookId,
        String liveBookName,
        String filter,
        List<String> fields
```

Сервис создания LiveBook:

```java
package com.pivotService.livebook;

import com.fasterxml.jackson.databind.JsonNode;
import com.pivotService.livebook.schema.LiveBookSchema;
import com.pivotService.livebook.schema.LiveBookSchemaBuilder;
import com.pivotService.livebook.schema.LiveBookSchemaRegistry;
import org.springframework.stereotype.Service;

@Service
public class LiveBookCreateService {

    private final LiveBookRequestValidator validator;
    private final LiveBookSchemaBuilder schemaBuilder;
    private final LiveBookSchemaRegistry schemaRegistry;
    private final LiveBookClient liveBookClient;

    public LiveBookCreateService(
            LiveBookRequestValidator validator,
            LiveBookSchemaBuilder schemaBuilder,
            LiveBookSchemaRegistry schemaRegistry,
            LiveBookClient liveBookClient
    ) {
        this.validator = validator;
        this.schemaBuilder = schemaBuilder;
        this.schemaRegistry = schemaRegistry;
        this.liveBookClient = liveBookClient;
    }

    public JsonNode create(CreateLiveBookRequest request) {
        validator.validate(request);

        LiveBookSchema schema = schemaBuilder.build(
                request.liveBookId(),
                request.liveBookName(),
                request.fields()
        );

        JsonNode response = liveBookClient.create(request);

        /*
         * Лучше регистрировать после успешного создания LiveBook в другом сервисе.
         */
        schemaRegistry.register(schema);

        return response;
    }
}
```

# 12. Как использовать в `ViewConfigValidator`

Самое важное изменение:  
`ViewConfigValidator` должен искать поля **не в `BusinessMetadataRegistry`**, а в `LiveBookSchema`.

То есть вместо:

```java
registry.findGlobalField(fieldKey)
```

нужно:

```java
liveBookSchema.findColumn(fieldKey)
```

---

## Обновите `SqlFieldRef`

Например так:

```java
package com.pivotService.SQLBuilder.validation;

import com.pivotService.metadata.FieldKind;
import com.pivotService.metadata.dto.FieldRestriction;

import java.util.List;

public record SqlFieldRef(
        String requestKey,
        String columnName,
        FieldKind kind,
        String rawType,
        List<FieldRestriction> restrictions
) {
    public SqlFieldRef {
        restrictions = restrictions == null ? List.of() : List.copyOf(restrictions);
    }
}
```

## В `ValidatedViewConfig` можно добавить `LiveBookSchema`

```java
package com.pivotService.SQLBuilder.validation;

import com.fasterxml.jackson.databind.JsonNode;
import com.pivotService.livebook.schema.LiveBookSchema;

import java.util.Map;

public final class ValidatedViewConfig {

    private final JsonNode config;
    private final String tableName;
    private final LiveBookSchema liveBookSchema;
    private final Map<String, SqlFieldRef> fieldsByRequestKey;

    public ValidatedViewConfig(
            JsonNode config,
            String tableName,
            LiveBookSchema liveBookSchema,
            Map<String, SqlFieldRef> fieldsByRequestKey
    ) {
        this.config = config;
        this.tableName = tableName;
        this.liveBookSchema = liveBookSchema;
        this.fieldsByRequestKey = Map.copyOf(fieldsByRequestKey);
    }

    public JsonNode config() {
        return config;
    }

    public String tableName() {
        return tableName;
    }

    public LiveBookSchema liveBookSchema() {
        return liveBookSchema;
    }

    public SqlFieldRef field(String fieldKey) {
        SqlFieldRef field = fieldsByRequestKey.get(fieldKey);

        if (field == null) {
            throw new IllegalArgumentException("Field was not validated: " + fieldKey);
        }

        return field;
    }

    public Map<String, SqlFieldRef> fieldsByRequestKey() {
        return fieldsByRequestKey;
    }
}
```

## Главное изменение в `ViewConfigValidator`

В начале `validate`:

```java
String liveBookName = resolveTableName(config);

LiveBookSchema liveBookSchema = liveBookSchemaRegistry.requireByName(liveBookName);
```

И дальше все поля резолвятся через `liveBookSchema`.

Пример метода `resolveField`:

```java
private SqlFieldRef resolveField(
        LiveBookSchema liveBookSchema,
        String fieldKey,
        String path,
        Map<String, SqlFieldRef> fields,
        ValidationErrors errors
) {
    if (fieldKey == null || fieldKey.isBlank()) {
        errors.reject(path, "FIELD_KEY_EMPTY", "fieldKey must not be empty");
        return null;
    }

    SqlFieldRef existing = fields.get(fieldKey);

    if (existing != null) {
        return existing;
    }

    LiveBookColumn column = liveBookSchema.findColumn(fieldKey).orElse(null);

    if (column == null) {
        errors.reject(
                path,
                "UNKNOWN_LIVEBOOK_FIELD",
                "Unknown fieldKey '" + fieldKey +
                        "' in LiveBook '" + liveBookSchema.liveBookName() +
                        "'. Available columns: " +
                        liveBookSchema.columns()
                                .stream()
                                .map(LiveBookColumn::columnName)
                                .toList()
        );
        return null;
    }

    SqlFieldRef ref = new SqlFieldRef(
            fieldKey,
            column.columnName(),
            column.kind(),
            column.rawType(),
            column.restrictions()
    );

    fields.put(fieldKey, ref);

    return ref;
}
```

Теперь `fieldKey = "base_amount"` будет найден в схеме LiveBook и преобразован в:

```text
requestKey = base_amount
columnName = baseAmount
type       = BIG_DECIMAL
```

SQL будет:

```sql
"baseAmount" AS "base_amount"
```

# 13. В SQL-builder важно правильно квотить columnName

Если колонка может содержать точку, например:

```text
alternativeIdentifiers.sourceId
```

нельзя делать так:

```sql
"alternativeIdentifiers"."sourceId"
```

Нужно так:

```sql
"alternativeIdentifiers.sourceId"
```

Поэтому лучше разделить методы:

```java
String quoteColumn(String columnName);
String quoteTable(String tableName);
```

Для колонок:

```java
public String quoteColumn(String columnName) {
    return "\"" + columnName.replace("\"", "\"\"") + "\"";
}
```

Для таблиц:

```java
public String quoteTable(String tableName) {
    String[] parts = tableName.split("\\.");

    StringJoiner joiner = new StringJoiner(".");

    for (String part : parts) {
        joiner.add("\"" + part.replace("\"", "\"\"") + "\"");
    }

    return joiner.toString();
}
```

И в SQL-builder:

```java
private String columnExpression(SqlFieldRef field) {
    return dialect.quoteColumn(field.columnName());
}
```

А не:

```java
dialect.quoteIdentifier(field.columnName())
```

если `quoteIdentifier` режет строку по точке.

# 14. Что получится для вашего примера

Запрос создания:

```json
{
  "liveBookId": 250,
  "liveBookName": "LiveBook250",
  "filter": "Trade.portfolioId==6052",
  "fields": [
    "Entity.id",
    "FxSpotForwardTrade.baseAmount",
    "FxSpotForwardTrade.baseCurrency.code",
    "Trade.portfolioId"
  ]

```

Схема LiveBook:

```text
LiveBook250 / livebook250

aliases:
  id                  -> id
  baseAmount          -> baseAmount
  base_amount         -> baseAmount
  baseCurrency_code   -> baseCurrency_code
  base_currency_code  -> baseCurrency_code
  portfolioId         -> portfolioId
  portfolio_id        -> portfolioId
```

Если потом `SetViewConfiguration` приходит так:

```json

  "liveBookName": "LiveBook250",
  "columns": [
    {
      "fieldKey": "base_amount"
    },
    {
      "fieldKey": "portfolio_id"
    }
  ]
}
```

SQL должен стать:

```sql
SELECT
  "baseAmount" AS "base_amount",
  "portfolioId" AS "portfolio_id",
  "id" AS "id"
FROM "livebook250"
LIMIT 100
```

# 15. Почему это решает `AMBIGUOUS_FIELD`

Раньше:

```text
instrument_id
```

искался глобально:

```text
FixingSource.instrument_id type=STRING
Trade.instrument_id type=LONG
TradeSaveEvent.instrument_id type=LONG
TradeFixingEvent.instrument_id type=LONG
```

И поэтому появлялась ошибка.

Теперь `instrument_id` ищется внутри конкретного LiveBook:

```text
LiveBook250 fields:
  Trade.instrumentId -> instrumentId LONG
```

То есть локальная схема знает:

```text
instrument_id -> instrumentId LONG
```

И никакого `FixingSource.instrument_id` в этой схеме вообще нет.

# 16. Главное правило

Для разных задач должны быть разные источники валидации:

| Задача | Где искать поле |
|---|---|
| Создание LiveBook | В иерархии бизнес-классов по class-qualified path |
| SQL к LiveBook-таблице | В схеме конкретного LiveBook |
| Диагностика метаданных | В глобальном `BusinessMetadataRegistry` |

То есть `BusinessMetadataRegistry.findGlobalField()` можно оставить для диагностики, но **не использовать его для SQL к конкретной LiveBook-таблице**.
