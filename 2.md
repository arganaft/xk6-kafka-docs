Ниже — вариант архитектуры и базовая реализация, которая решает ваши задачи:

- строит индекс бизнес-классов из иерархического JSON;
- учитывает наследование полей;
- хранит поля в двух индексах: по `jsonName` и по `name`;
- валидирует запрос создания LiveBook;
- валидирует `SetViewConfiguration`;
- дает типы полей для SQL;
- рефакторит SQL-builder так, чтобы он работал не со “строками из запроса”, а с валидированными метаданными.

---

## 1. Общая идея

Вам нужен центральный слой метаданных:

```text
JSON hierarchy
   |
   v
BusinessMetadataRegistry
   |
   |-- class sourceValue -> BusinessClassMetadata
   |
   |-- внутри каждого класса:
       |-- fieldsByJsonName    // для LiveBook: FxSpotForwardTrade.baseAmount
       |-- fieldsByName        // для SQL config: base_amount
       |-- relationsByAlias    // для путей типа baseCurrency.code
```

Важно:

- `fieldsByJsonName` нужен для LiveBook-запроса.
- `fieldsByName` нужен для `SetViewConfiguration`, потому что там приходят `fieldKey` как `name`.
- SQL должен использовать `jsonName`.
- потомок получает унаследованные поля родителя.

# 3. Тип поля

## `FieldKind.java`

```java
package com.pivotService.metadata;

import java.util.Locale;

public enum FieldKind {
    STRING,
    INTEGER,
    LONG,
    BIG_DECIMAL,
    BOOLEAN,
    LOCAL_DATE,
    INSTANT,
    UUID;

    public static FieldKind from(String fieldType) {
        if (fieldType == null) {
            return STRING;
        }

        return switch (fieldType.toLowerCase(Locale.ROOT)) {
            case "string", "varchar", "text" -> STRING;
            case "integer", "int" -> INTEGER;
            case "long", "bigint" -> LONG;
            case "numeric", "decimal", "bigdecimal" -> BIG_DECIMAL;
            case "boolean", "bool" -> BOOLEAN;
            case "date", "localdate" -> LOCAL_DATE;
            case "datetime", "timestamp", "instant", "localdatetime" -> INSTANT;
            case "uuid" -> UUID;
            default -> STRING;
        };
    }

    public String javaType() {
        return switch (this) {
            case STRING -> "String";
            case INTEGER -> "Integer";
            case LONG -> "Long";
            case BIG_DECIMAL -> "BigDecimal";
            case BOOLEAN -> "Boolean";
            case LOCAL_DATE -> "LocalDate";
            case INSTANT -> "Instant";
            case UUID -> "UUID";
        };
    }

    public boolean isNumeric() {
        return this == INTEGER || this == LONG || this == BIG_DECIMAL;
    }

    public boolean isDateLike() {
        return this == LOCAL_DATE || this == INSTANT;
    }

    public boolean isTextLike() {
        return this == STRING || this == UUID;
    }

    public boolean isComparable() {
        return isNumeric() || isDateLike();
    }
}
```

# 5. Индекс всех бизнес-классов

## `BusinessMetadataRegistry.java`

```java
package com.pivotService.metadata;

import com.pivotService.metadata.dto.BusinessClassNode;
import com.pivotService.metadata.dto.BusinessField;
import com.pivotService.metadata.dto.BusinessRelation;

import java.util.*;

public final class BusinessMetadataRegistry {

    private final Map<String, BusinessClassMetadata> classesBySourceValue;
    private final Map<String, String> classAliases;

    private BusinessMetadataRegistry(
            Map<String, BusinessClassMetadata> classesBySourceValue,
            Map<String, String> classAliases
    ) {
        this.classesBySourceValue = Map.copyOf(classesBySourceValue);
        this.classAliases = Map.copyOf(classAliases);
    }

    public static BusinessMetadataRegistry empty() {
        return new BusinessMetadataRegistry(Map.of(), Map.of());
    }

    public static BusinessMetadataRegistry from(Collection<BusinessClassNode> roots) {
        Map<String, BusinessClassMetadata> classes = new LinkedHashMap<>();
        Map<String, String> aliases = new HashMap<>();

        for (BusinessClassNode root : roots) {
            indexNode(
                    root,
                    new LinkedHashMap<>(),
                    new LinkedHashMap<>(),
                    new LinkedHashMap<>(),
                    List.of(),
                    classes,
                    aliases
            );
        }

        return new BusinessMetadataRegistry(classes, aliases);
    }

    public boolean isEmpty() {
        return classesBySourceValue.isEmpty();
    }

    public Optional<BusinessClassMetadata> findClass(String classKey) {
        if (!hasText(classKey)) {
            return Optional.empty();
        }

        BusinessClassMetadata exact = classesBySourceValue.get(classKey);
        if (exact != null) {
            return Optional.of(exact);
        }

        String sourceValue = classAliases.get(classKey);
        if (sourceValue == null) {
            sourceValue = classAliases.get(classKey.toLowerCase(Locale.ROOT));
        }

        if (sourceValue == null) {
            return Optional.empty();
        }

        return Optional.ofNullable(classesBySourceValue.get(sourceValue));
    }

    public BusinessClassMetadata requireClass(String classKey) {
        return findClass(classKey)
                .orElseThrow(() -> new IllegalArgumentException("Unknown business class: " + classKey));
    }

    private static void indexNode(
            BusinessClassNode node,
            Map<String, FieldMetadata> inheritedByJsonName,
            Map<String, FieldMetadata> inheritedByName,
            Map<String, BusinessRelation> inheritedRelationsByAlias,
            List<String> ancestors,
            Map<String, BusinessClassMetadata> classes,
            Map<String, String> aliases
    ) {
        String sourceValue = hasText(node.sourceValue()) ? node.sourceValue() : node.name();

        Map<String, FieldMetadata> fieldsByJsonName = new LinkedHashMap<>(inheritedByJsonName);
        Map<String, FieldMetadata> fieldsByName = new LinkedHashMap<>(inheritedByName);

        for (BusinessField field : node.fields()) {
            FieldMetadata metadata = new FieldMetadata(
                    sourceValue,
                    field,
                    FieldKind.from(field.type())
            );

            if (hasText(field.effectiveJsonName())) {
                fieldsByJsonName.put(field.effectiveJsonName(), metadata);
            }

            if (hasText(field.effectiveName())) {
                fieldsByName.put(field.effectiveName(), metadata);
            }
        }

        Map<String, BusinessRelation> relationsByAlias = new LinkedHashMap<>(inheritedRelationsByAlias);

        for (BusinessRelation relation : node.relations()) {
            if (hasText(relation.alias())) {
                relationsByAlias.put(relation.alias(), relation);
            }
        }

        BusinessClassMetadata classMetadata = new BusinessClassMetadata(
                node.name(),
                sourceValue,
                fieldsByJsonName,
                fieldsByName,
                relationsByAlias,
                ancestors
        );

        classes.put(sourceValue, classMetadata);

        putAlias(aliases, sourceValue, sourceValue);
        putAlias(aliases, node.name(), sourceValue);

        List<String> childAncestors = new ArrayList<>(ancestors);
        childAncestors.add(sourceValue);

        for (BusinessClassNode child : node.children()) {
            indexNode(
                    child,
                    fieldsByJsonName,
                    fieldsByName,
                    relationsByAlias,
                    childAncestors,
                    classes,
                    aliases
            );
        }
    }

    private static void putAlias(Map<String, String> aliases, String alias, String sourceValue) {
        if (!hasText(alias)) {
            return;
        }

        aliases.put(alias, sourceValue);
        aliases.put(alias.toLowerCase(Locale.ROOT), sourceValue);
    }

    private static boolean hasText(String value) {
        return value != null && !value.isBlank();
    }
}
```

# 6. Сервис хранения актуальных метаданных

Ваш контроллер, который получает JSON-иерархию, должен вызвать `refresh`.

## `BusinessMetadataService.java`

```java
package com.pivotService.metadata;

import com.fasterxml.jackson.core.type.TypeReference;
import com.fasterxml.jackson.databind.JsonNode;
import com.fasterxml.jackson.databind.ObjectMapper;
import com.pivotService.metadata.dto.BusinessClassNode;
import org.springframework.stereotype.Service;

import java.util.List;
import java.util.concurrent.atomic.AtomicReference;

@Service
public class BusinessMetadataService {

    private final ObjectMapper objectMapper;
    private final AtomicReference<BusinessMetadataRegistry> registryRef =
            new AtomicReference<>(BusinessMetadataRegistry.empty());

    public BusinessMetadataService(ObjectMapper objectMapper) {
        this.objectMapper = objectMapper;
    }

    public void refresh(JsonNode hierarchyJson) {
        try {
            List<BusinessClassNode> roots;

            if (hierarchyJson.isArray()) {
                roots = objectMapper.convertValue(
                        hierarchyJson,
                        new TypeReference<List<BusinessClassNode>>() {
                        }
                );
            } else {
                roots = List.of(objectMapper.treeToValue(hierarchyJson, BusinessClassNode.class));
            }

            registryRef.set(BusinessMetadataRegistry.from(roots));
        } catch (Exception e) {
            throw new IllegalArgumentException("Cannot parse business metadata hierarchy", e);
        }
    }

    public BusinessMetadataRegistry registry() {
        BusinessMetadataRegistry registry = registryRef.get();

        if (registry.isEmpty()) {
            throw new IllegalStateException("Business metadata registry is empty");
        }

        return registry;
    }
}
```

Пример использования в вашем контроллере метаданных:

```java
@PostMapping("/metadata")
public ResponseEntity<Void> updateMetadata(@RequestBody JsonNode body) {
    businessMetadataService.refresh(body);
    return ResponseEntity.ok().build();
}
```

## `BusinessPathResolver.java`

```java
package com.pivotService.metadata;

import com.pivotService.metadata.dto.BusinessRelation;
import org.springframework.stereotype.Component;

import java.util.ArrayList;
import java.util.List;

@Component
public class BusinessPathResolver {

    private final BusinessMetadataService metadataService;

    public BusinessPathResolver(BusinessMetadataService metadataService) {
        this.metadataService = metadataService;
    }

    public ResolvedFieldPath resolveByJsonPath(String rawPath) {
        if (rawPath == null || rawPath.isBlank()) {
            throw new IllegalArgumentException("Field path is empty");
        }

        String[] parts = rawPath.split("\\.");

        if (parts.length < 2) {
            throw new IllegalArgumentException(
                    "Field path must have format Class.field or Class.relation.field: " + rawPath
            );
        }

        BusinessMetadataRegistry registry = metadataService.registry();

        BusinessClassMetadata startClass = registry.findClass(parts[0])
                .orElseThrow(() -> new IllegalArgumentException(
                        "Unknown class '" + parts[0] + "' in path '" + rawPath + "'"
                ));

        BusinessClassMetadata currentClass = startClass;
        List<BusinessRelation> relations = new ArrayList<>();

        for (int i = 1; i < parts.length; i++) {
            String segment = parts[i];
            boolean last = i == parts.length - 1;

            if (last) {
                FieldMetadata field = currentClass.findFieldByJsonName(segment)
                        .orElseThrow(() -> new IllegalArgumentException(
                                "Unknown json field '" + segment + "' in class '" +
                                        currentClass.sourceValue() + "' for path '" + rawPath + "'"
                        ));

                return new ResolvedFieldPath(
                        rawPath,
                        startClass,
                        currentClass,
                        field,
                        List.copyOf(relations)
                );
            }

            BusinessRelation relation = currentClass.findRelationByAlias(segment)
                    .orElseThrow(() -> new IllegalArgumentException(
                            "Unknown relation alias '" + segment + "' in class '" +
                                    currentClass.sourceValue() + "' for path '" + rawPath + "'"
                    ));

            if (relation.targetClass() == null || relation.targetClass().isBlank()) {
                throw new IllegalArgumentException(
                        "Relation alias '" + segment + "' has empty targetClass in path '" + rawPath + "'"
                );
            }

            currentClass = registry.findClass(relation.targetClass())
                    .orElseThrow(() -> new IllegalArgumentException(
                            "Unknown targetClass '" + relation.targetClass() +
                                    "' for relation alias '" + segment + "' in path '" + rawPath + "'"
                    ));

            relations.add(relation);
        }

        throw new IllegalArgumentException("Invalid field path: " + rawPath);
    }

    public FieldMetadata resolveFieldKey(String rootClass, String fieldKey) {
        BusinessClassMetadata root = metadataService.registry().requireClass(rootClass);

        return root.findFieldByName(fieldKey)
                .or(() -> root.findFieldByJsonName(fieldKey))
                .orElseThrow(() -> new IllegalArgumentException(
                        "Unknown fieldKey '" + fieldKey + "' in class '" + rootClass + "'"
                ));
    }
}
```

# 9. Валидация значения по типу и restrictions

## `FieldValueValidator.java`

```java
package com.pivotService.validation;

import com.pivotService.metadata.FieldKind;
import com.pivotService.metadata.FieldMetadata;
import com.pivotService.metadata.dto.EnumRestrictionValue;
import com.pivotService.metadata.dto.FieldRestriction;
import org.springframework.stereotype.Component;

import java.math.BigDecimal;
import java.time.Instant;
import java.time.LocalDate;
import java.time.LocalDateTime;
import java.time.ZoneOffset;
import java.time.format.DateTimeParseException;
import java.util.Locale;
import java.util.Set;
import java.util.UUID;
import java.util.stream.Collectors;

@Component
public class FieldValueValidator {

    public void validateLiteral(
            FieldMetadata field,
            String rawValue,
            String path,
            ValidationErrors errors
    ) {
        if (rawValue == null) {
            return;
        }

        String value = unquote(rawValue.trim());

        if ("null".equalsIgnoreCase(value)) {
            return;
        }

        validateType(field, value, path, errors);

        if (!errors.hasErrors()) {
            validateRestrictions(field, value, path, errors);
        }
    }

    private void validateType(
            FieldMetadata field,
            String value,
            String path,
            ValidationErrors errors
    ) {
        try {
            FieldKind kind = field.kind();

            switch (kind) {
                case STRING -> {
                    // Любая строка допустима, ограничения проверяются отдельно.
                }
                case INTEGER -> Integer.parseInt(value);
                case LONG -> Long.parseLong(value);
                case BIG_DECIMAL -> new BigDecimal(value);
                case BOOLEAN -> {
                    String normalized = value.toLowerCase(Locale.ROOT);

                    if (!Set.of("true", "false", "1", "0").contains(normalized)) {
                        throw new IllegalArgumentException("Expected boolean value");
                    }
                }
                case LOCAL_DATE -> LocalDate.parse(value);
                case INSTANT -> parseInstantLike(value);
                case UUID -> UUID.fromString(value);
            }
        } catch (Exception e) {
            errors.reject(
                    path,
                    "INVALID_VALUE_TYPE",
                    "Value '" + value + "' does not match field '" +
                            field.jsonName() + "' type '" + field.kind().javaType() + "'"
            );
        }
    }

    private void validateRestrictions(
            FieldMetadata field,
            String value,
            String path,
            ValidationErrors errors
    ) {
        for (FieldRestriction restriction : field.field().restrictions()) {
            if (restriction.type() == null) {
                continue;
            }

            String type = restriction.type().toUpperCase(Locale.ROOT);

            switch (type) {
                case "ENUM" -> validateEnumRestriction(field, value, restriction, path, errors);
                case "MAX_LENGTH" -> validateMaxLengthRestriction(field, value, restriction, path, errors);
                default -> {
                    // Неизвестные restrictions можно логировать, но не обязательно падать.
                }
            }
        }
    }

    private void validateEnumRestriction(
            FieldMetadata field,
            String value,
            FieldRestriction restriction,
            String path,
            ValidationErrors errors
    ) {
        Set<String> allowed = restriction.values()
                .stream()
                .map(EnumRestrictionValue::value)
                .collect(Collectors.toSet());

        if (!allowed.contains(value)) {
            errors.reject(
                    path,
                    "ENUM_VIOLATION",
                    "Field '" + field.jsonName() + "' allows only values " + allowed +
                            ", but got '" + value + "'"
            );
        }
    }

    private void validateMaxLengthRestriction(
            FieldMetadata field,
            String value,
            FieldRestriction restriction,
            String path,
            ValidationErrors errors
    ) {
        Integer max = restriction.value();

        if (max != null && value.length() > max) {
            errors.reject(
                    path,
                    "MAX_LENGTH_VIOLATION",
                    "Field '" + field.jsonName() + "' max length is " + max +
                            ", but got length " + value.length()
            );
        }
    }

    private void parseInstantLike(String value) {
        try {
            Instant.parse(value);
            return;
        } catch (DateTimeParseException ignored) {
            // Пробуем LocalDateTime ниже.
        }

        LocalDateTime.parse(value).atZone(ZoneOffset.UTC).toInstant();
    }

    private String unquote(String value) {
        if (value.length() >= 2) {
            boolean singleQuoted = value.startsWith("'") && value.endsWith("'");
            boolean doubleQuoted = value.startsWith("\"") && value.endsWith("\"");

            if (singleQuoted || doubleQuoted) {
                return value.substring(1, value.length() - 1);
            }
        }

        return value;
    }
}
```

```java
import org.springframework.stereotype.Component;

import java.util.ArrayList;
import java.util.List;
import java.util.regex.Matcher;
import java.util.regex.Pattern;

@Component
public class LiveBookFilterParser {

    private static final Pattern SIMPLE_CONDITION = Pattern.compile(
            "^\\s*(?<path>[A-Za-z_][A-Za-z0-9_]*(?:\\.[A-Za-z_][A-Za-z0-9_]*)+)\\s*" +
                    "(?<op>==|!=|>=|<=|>|<)\\s*" +
                    "(?<value>.+?)\\s*$"
    );

    public List<ParsedFilterCondition> parse(String filter) {
        if (filter == null || filter.isBlank()) {
            return List.of();
        }

        List<String> conditionStrings = splitByLogicalOperators(filter);
        List<ParsedFilterCondition> result = new ArrayList<>();

        for (String conditionString : conditionStrings) {
            Matcher matcher = SIMPLE_CONDITION.matcher(conditionString);

            if (!matcher.matches()) {
                throw new IllegalArgumentException("Invalid filter condition: " + conditionString);
            }

            result.add(new ParsedFilterCondition(
                    matcher.group("path"),
                    matcher.group("op"),
                    matcher.group("value")
            ));
        }

        return result;
    }

    private List<String> splitByLogicalOperators(String filter) {
        // Базовый вариант. Если фильтры станут сложными, лучше перейти на RSQL/FIQL/parser combinators.
        return List.of(filter.split("(?i)\\s+(?:AND|OR)\\s+"));
    }
}
```

```java
package com.pivotService.livebook;

import com.pivotService.metadata.BusinessPathResolver;
import com.pivotService.metadata.FieldKind;
import com.pivotService.metadata.ResolvedFieldPath;
import com.pivotService.validation.FieldValueValidator;
import com.pivotService.validation.ValidationErrors;
import org.springframework.stereotype.Service;

import java.util.List;
import java.util.Set;

@Service
public class LiveBookRequestValidator {

    private static final Set<String> EQUALITY_OPERATORS = Set.of("==", "!=");
    private static final Set<String> ORDER_OPERATORS = Set.of(">", ">=", "<", "<=");

    private final BusinessPathResolver pathResolver;
    private final LiveBookFilterParser filterParser;
    private final FieldValueValidator fieldValueValidator;

    public LiveBookRequestValidator(
            BusinessPathResolver pathResolver,
            LiveBookFilterParser filterParser,
            FieldValueValidator fieldValueValidator
    ) {
        this.pathResolver = pathResolver;
        this.filterParser = filterParser;
        this.fieldValueValidator = fieldValueValidator;
    }

    public void validate(CreateLiveBookRequest request) {
        ValidationErrors errors = new ValidationErrors();

        if (request == null) {
            errors.reject("$", "REQUEST_NULL", "Request body is required");
            errors.throwIfAny();
        }

        validateFields(request.fields(), errors);
        validateFilter(request.filter(), errors);

        errors.throwIfAny();
    }

    private void validateFields(List<String> fields, ValidationErrors errors) {
        if (fields == null || fields.isEmpty()) {
            errors.reject("$.fields", "FIELDS_EMPTY", "fields must not be empty");
            return;
        }

        for (int i = 0; i < fields.size(); i++) {
            String fieldPath = fields.get(i);

            if (fieldPath == null || fieldPath.isBlank()) {
                errors.reject("$.fields[" + i + "]", "FIELD_EMPTY", "Field path must not be empty");
                continue;
            }

            try {
                pathResolver.resolveByJsonPath(fieldPath);
            } catch (Exception e) {
                errors.reject("$.fields[" + i + "]", "UNKNOWN_FIELD_PATH", e.getMessage());
            }
        }
    }

    private void validateFilter(String filter, ValidationErrors errors) {
        List<ParsedFilterCondition> conditions;

        try {
            conditions = filterParser.parse(filter);
        } catch (Exception e) {
            errors.reject("$.filter", "INVALID_FILTER_SYNTAX", e.getMessage());
            return;
        }

        for (ParsedFilterCondition condition : conditions) {
            ResolvedFieldPath resolved;

            try {
                resolved = pathResolver.resolveByJsonPath(condition.fieldPath());
            } catch (Exception e) {
                errors.reject("$.filter", "UNKNOWN_FILTER_FIELD", e.getMessage());
                continue;
            }

            validateOperator(resolved, condition.operator(), errors);

            fieldValueValidator.validateLiteral(
                    resolved.field(),
                    condition.rawValue(),
                    "$.filter",
                    errors
            );
        }
    }

    private void validateOperator(
            ResolvedFieldPath resolved,
            String operator,
            ValidationErrors errors
    ) {
        FieldKind kind = resolved.field().kind();

        if (kind == FieldKind.BOOLEAN && !EQUALITY_OPERATORS.contains(operator)) {
            errors.reject(
                    "$.filter",
                    "INVALID_OPERATOR",
                    "Boolean field '" + resolved.originalPath() +
                            "' supports only operators " + EQUALITY_OPERATORS
            );
        }

        if (kind == FieldKind.STRING && ORDER_OPERATORS.contains(operator)) {
            errors.reject(
                    "$.filter",
                    "INVALID_OPERATOR",
                    "String field '" + resolved.originalPath() +
                            "' does not support operator '" + operator + "'"
            );
        }
    }
}
```

```java
package com.pivotService.SQLBuilder.validation;

import com.fasterxml.jackson.databind.JsonNode;
import com.pivotService.metadata.BusinessClassMetadata;
import com.pivotService.metadata.BusinessMetadataService;
import com.pivotService.metadata.FieldKind;
import com.pivotService.metadata.FieldMetadata;
import com.pivotService.validation.FieldValueValidator;
import com.pivotService.validation.ValidationErrors;
import org.springframework.beans.factory.annotation.Value;
import org.springframework.stereotype.Service;

import java.util.Iterator;
import java.util.LinkedHashMap;
import java.util.Locale;
import java.util.Map;
import java.util.Set;

@Service
public class ViewConfigValidator {

    private static final Set<String> AGG_FUNCS =
            Set.of("sum", "avg", "min", "max", "count", "unique_count");

    private static final Set<String> SORT_DIRECTIONS =
            Set.of("asc", "desc");

    private final BusinessMetadataService metadataService;
    private final FieldValueValidator fieldValueValidator;

    @Value("${app.liveBook-name:livebook_FxSpotForwardTrade}")
    private String defaultTableName;

    @Value("${app.default-root-class:FxSpotForwardTrade}")
    private String defaultRootClass;

    @Value("${app.livebook-table-prefix:livebook_}")
    private String liveBookTablePrefix;

    public ViewConfigValidator(
            BusinessMetadataService metadataService,
            FieldValueValidator fieldValueValidator
    ) {
        this.metadataService = metadataService;
        this.fieldValueValidator = fieldValueValidator;
    }

    public ValidatedViewConfig validate(JsonNode config) {
        ValidationErrors errors = new ValidationErrors();

        if (config == null || !config.isObject()) {
            errors.reject("$", "INVALID_CONFIG", "Config must be JSON object");
            errors.throwIfAny();
        }

        String tableName = resolveTableName(config);
        String rootClass = resolveRootClass(config, tableName);

        BusinessClassMetadata rootMetadata = metadataService.registry()
                .findClass(rootClass)
                .orElse(null);

        if (rootMetadata == null) {
            errors.reject(
                    "$.rootClass",
                    "UNKNOWN_ROOT_CLASS",
                    "Unknown root class: " + rootClass
            );
            errors.throwIfAny();
        }

        Map<String, SqlFieldRef> fields = new LinkedHashMap<>();

        validatePaging(config, errors);

        validateFieldArray(config, "columns", rootMetadata, fields, errors);
        validateFieldArray(config, "rowGroupColumns", rootMetadata, fields, errors);
        validateFieldArray(config, "pivotColumns", rootMetadata, fields, errors);
        validateValueColumns(config, rootMetadata, fields, errors);
        validateSortModel(config, rootMetadata, fields, errors);
        validateGroupKeys(config, rootMetadata, fields, errors);
        validateFilterModel(config.path("filterModel"), rootMetadata, fields, errors);

        // Ваш старый builder всегда добавляет id.
        // Поэтому id тоже должен быть валидным полем.
        resolveField(rootMetadata, "id", "$.columns[id:auto]", fields, errors);

        errors.throwIfAny();

        return new ValidatedViewConfig(
                config,
                tableName,
                rootMetadata.sourceValue(),
                fields
        );
    }

    private String resolveTableName(JsonNode config) {
        if (config.hasNonNull("liveBookName")) {
            return config.get("liveBookName").asText();
        }

        return defaultTableName;
    }

    private String resolveRootClass(JsonNode config, String tableName) {
        if (config.hasNonNull("rootClass")) {
            return config.get("rootClass").asText();
        }

        if (config.hasNonNull("sourceValue")) {
            return config.get("sourceValue").asText();
        }

        if (tableName != null && tableName.startsWith(liveBookTablePrefix)) {
            return tableName.substring(liveBookTablePrefix.length());
        }

        return defaultRootClass;
    }

    private void validatePaging(JsonNode config, ValidationErrors errors) {
        int startRow = config.path("startRow").asInt(0);
        int endRow = config.path("endRow").asInt(100);

        if (startRow < 0) {
            errors.reject("$.startRow", "INVALID_START_ROW", "startRow must be greater than or equal to 0");
        }

        if (endRow < startRow) {
            errors.reject("$.endRow", "INVALID_END_ROW", "endRow must be greater than or equal to startRow");
        }
    }

    private void validateFieldArray(
            JsonNode config,
            String arrayName,
            BusinessClassMetadata rootMetadata,
            Map<String, SqlFieldRef> fields,
            ValidationErrors errors
    ) {
        JsonNode array = config.path(arrayName);

        if (array.isMissingNode() || array.isNull()) {
            return;
        }

        if (!array.isArray()) {
            errors.reject("$." + arrayName, "INVALID_ARRAY", arrayName + " must be array");
            return;
        }

        for (int i = 0; i < array.size(); i++) {
            JsonNode item = array.get(i);
            String fieldKey = item.path("fieldKey").asText(null);

            SqlFieldRef field = resolveField(
                    rootMetadata,
                    fieldKey,
                    "$." + arrayName + "[" + i + "].fieldKey",
                    fields,
                    errors
            );

            validateGroupByIfPresent(
                    item,
                    field,
                    "$." + arrayName + "[" + i + "]",
                    errors
            );
        }
    }

    private void validateValueColumns(
            JsonNode config,
            BusinessClassMetadata rootMetadata,
            Map<String, SqlFieldRef> fields,
            ValidationErrors errors
    ) {
        JsonNode valueColumns = config.path("valueColumns");

        if (valueColumns.isMissingNode() || valueColumns.isNull()) {
            return;
        }

        if (!valueColumns.isArray()) {
            errors.reject("$.valueColumns", "INVALID_ARRAY", "valueColumns must be array");
            return;
        }

        for (int i = 0; i < valueColumns.size(); i++) {
            JsonNode item = valueColumns.get(i);
            String fieldKey = item.path("fieldKey").asText(null);
            String aggFunc = item.path("aggFunc").asText(null);

            SqlFieldRef field = resolveField(
                    rootMetadata,
                    fieldKey,
                    "$.valueColumns[" + i + "].fieldKey",
                    fields,
                    errors
            );

            validateAggFunc(
                    field,
                    aggFunc,
                    "$.valueColumns[" + i + "].aggFunc",
                    errors
            );
        }
    }

    private void validateSortModel(
            JsonNode config,
            BusinessClassMetadata rootMetadata,
            Map<String, SqlFieldRef> fields,
            ValidationErrors errors
    ) {
        JsonNode sortModel = config.path("sortModel");

        if (sortModel.isMissingNode() || sortModel.isNull()) {
            return;
        }

        if (!sortModel.isArray()) {
            errors.reject("$.sortModel", "INVALID_ARRAY", "sortModel must be array");
            return;
        }

        for (int i = 0; i < sortModel.size(); i++) {
            JsonNode item = sortModel.get(i);

            String colId = item.path("colId").asText(null);
            String sort = item.path("sort").asText("asc").toLowerCase(Locale.ROOT);

            resolveField(
                    rootMetadata,
                    colId,
                    "$.sortModel[" + i + "].colId",
                    fields,
                    errors
            );

            if (!SORT_DIRECTIONS.contains(sort)) {
                errors.reject(
                        "$.sortModel[" + i + "].sort",
                        "INVALID_SORT_DIRECTION",
                        "sort must be one of " + SORT_DIRECTIONS
                );
            }
        }
    }

    private void validateGroupKeys(
            JsonNode config,
            BusinessClassMetadata rootMetadata,
            Map<String, SqlFieldRef> fields,
            ValidationErrors errors
    ) {
        JsonNode groupKeys = config.path("groupKeys");

        if (groupKeys.isMissingNode() || groupKeys.isNull()) {
            return;
        }

        if (!groupKeys.isArray()) {
            errors.reject("$.groupKeys", "INVALID_ARRAY", "groupKeys must be array");
            return;
        }

        for (int i = 0; i < groupKeys.size(); i++) {
            JsonNode item = groupKeys.get(i);

            String fieldKey = item.path("fieldKey").asText(null);
            SqlFieldRef field = resolveField(
                    rootMetadata,
                    fieldKey,
                    "$.groupKeys[" + i + "].fieldKey",
                    fields,
                    errors
            );

            if (item.has("value") && field != null) {
                fieldValueValidator.validateLiteral(
                        field.metadata(),
                        item.get("value").asText(),
                        "$.groupKeys[" + i + "].value",
                        errors
                );
            }
        }
    }

    private void validateFilterModel(
            JsonNode filterModel,
            BusinessClassMetadata rootMetadata,
            Map<String, SqlFieldRef> fields,
            ValidationErrors errors
    ) {
        if (filterModel == null || filterModel.isMissingNode() || filterModel.isNull()) {
            return;
        }

        if (!filterModel.isObject()) {
            errors.reject("$.filterModel", "INVALID_OBJECT", "filterModel must be object");
            return;
        }

        Iterator<Map.Entry<String, JsonNode>> iterator = filterModel.fields();

        while (iterator.hasNext()) {
            Map.Entry<String, JsonNode> entry = iterator.next();

            String fieldKey = entry.getKey();
            SqlFieldRef field = resolveField(
                    rootMetadata,
                    fieldKey,
                    "$.filterModel." + fieldKey,
                    fields,
                    errors
            );

            if (field != null) {
                validateAgGridFilter(
                        field,
                        entry.getValue(),
                        "$.filterModel." + fieldKey,
                        errors
                );
            }
        }
    }

    private void validateAgGridFilter(
            SqlFieldRef field,
            JsonNode filterDef,
            String path,
            ValidationErrors errors
    ) {
        if (filterDef == null || filterDef.isMissingNode() || filterDef.isNull()) {
            return;
        }

        String filterType = filterDef.path("filterType").asText(null);

        validateFilterTypeMatchesField(field, filterType, path + ".filterType", errors);

        if (filterDef.has("operator")) {
            validateAgGridFilter(field, filterDef.path("condition1"), path + ".condition1", errors);
            validateAgGridFilter(field, filterDef.path("condition2"), path + ".condition2", errors);
            return;
        }

        if ("set".equalsIgnoreCase(filterType)) {
            validateSetFilter(field, filterDef, path, errors);
            return;
        }

        validateSingleCondition(field, filterDef, filterType, path, errors);
    }

    private void validateSingleCondition(
            SqlFieldRef field,
            JsonNode condition,
            String filterType,
            String path,
            ValidationErrors errors
    ) {
        String type = condition.path("type").asText("");

        switch (type) {
            case "equals", "notEqual", "lessThan", "lessThanOrEqual",
                 "greaterThan", "greaterThanOrEqual" -> {
                String value = extractFilterValue(condition, filterType, false);

                if (value == null) {
                    errors.reject(path, "FILTER_VALUE_REQUIRED", "Filter value is required");
                } else {
                    fieldValueValidator.validateLiteral(field.metadata(), value, path, errors);
                }
            }

            case "inRange" -> {
                String from = extractFilterValue(condition, filterType, false);
                String to = extractFilterValue(condition, filterType, true);

                fieldValueValidator.validateLiteral(field.metadata(), from, path + ".from", errors);
                fieldValueValidator.validateLiteral(field.metadata(), to, path + ".to", errors);
            }

            case "contains", "notContains", "startsWith", "endsWith" -> {
                if (!field.kind().isTextLike()) {
                    errors.reject(
                            path + ".type",
                            "INVALID_FILTER_OPERATION",
                            "Text operation '" + type + "' is not allowed for field type " + field.kind()
                    );
                }

                String value = condition.path("filter").asText(null);
                fieldValueValidator.validateLiteral(field.metadata(), value, path + ".filter", errors);
            }

            case "blank", "notBlank", "empty", "notEmpty" -> {
                // Значение не требуется.
            }

            default -> errors.reject(
                    path + ".type",
                    "UNKNOWN_FILTER_OPERATION",
                    "Unknown filter operation: " + type
            );
        }
    }

    private void validateSetFilter(
            SqlFieldRef field,
            JsonNode filterDef,
            String path,
            ValidationErrors errors
    ) {
        JsonNode values = filterDef.path("values");

        if (!values.isArray()) {
            errors.reject(path + ".values", "INVALID_VALUES", "set filter values must be array");
            return;
        }

        for (int i = 0; i < values.size(); i++) {
            if (!values.get(i).isNull()) {
                fieldValueValidator.validateLiteral(
                        field.metadata(),
                        values.get(i).asText(),
                        path + ".values[" + i + "]",
                        errors
                );
            }
        }
    }

    private void validateFilterTypeMatchesField(
            SqlFieldRef field,
            String filterType,
            String path,
            ValidationErrors errors
    ) {
        if (filterType == null || filterType.isBlank()) {
            return;
        }

        String normalized = filterType.toLowerCase(Locale.ROOT);

        switch (normalized) {
            case "number" -> {
                if (!field.kind().isNumeric()) {
                    errors.reject(path, "FILTER_TYPE_MISMATCH", "number filter is not allowed for " + field.kind());
                }
            }
            case "date" -> {
                if (!field.kind().isDateLike()) {
                    errors.reject(path, "FILTER_TYPE_MISMATCH", "date filter is not allowed for " + field.kind());
                }
            }
            case "text" -> {
                if (!field.kind().isTextLike()) {
                    errors.reject(path, "FILTER_TYPE_MISMATCH", "text filter is not allowed for " + field.kind());
                }
            }
            case "set" -> {
                // set допустим почти для всех типов, значения проверяются отдельно.
            }
            default -> errors.reject(path, "UNKNOWN_FILTER_TYPE", "Unknown filterType: " + filterType);
        }
    }

    private void validateGroupByIfPresent(
            JsonNode item,
            SqlFieldRef field,
            String path,
            ValidationErrors errors
    ) {
        if (field == null || !item.hasNonNull("groupBy")) {
            return;
        }

        String groupBy = item.get("groupBy").asText().toLowerCase(Locale.ROOT);

        if (!Set.of("year", "month", "day").contains(groupBy)) {
            errors.reject(
                    path + ".groupBy",
                    "INVALID_GROUP_BY",
                    "groupBy must be one of year, month, day"
            );
            return;
        }

        if (!field.kind().isDateLike()) {
            errors.reject(
                    path + ".groupBy",
                    "GROUP_BY_REQUIRES_DATE",
                    "Date groupBy requires DATE/DATETIME field, but got " + field.kind()
            );
        }
    }

    private void validateAggFunc(
            SqlFieldRef field,
            String aggFunc,
            String path,
            ValidationErrors errors
    ) {
        if (aggFunc == null || aggFunc.isBlank()) {
            errors.reject(path, "AGG_FUNC_REQUIRED", "aggFunc is required");
            return;
        }

        String normalized = aggFunc.toLowerCase(Locale.ROOT);

        if (!AGG_FUNCS.contains(normalized)) {
            errors.reject(path, "INVALID_AGG_FUNC", "aggFunc must be one of " + AGG_FUNCS);
            return;
        }

        if (field == null) {
            return;
        }

        if (Set.of("sum", "avg").contains(normalized) && !field.kind().isNumeric()) {
            errors.reject(
                    path,
                    "AGG_FUNC_REQUIRES_NUMERIC",
                    "Aggregation '" + normalized + "' requires numeric field, but got " + field.kind()
            );
        }
    }

    private SqlFieldRef resolveField(
            BusinessClassMetadata rootMetadata,
            String fieldKey,
            String path,
            Map<String, SqlFieldRef> fields,
            ValidationErrors errors
    ) {
        if (fieldKey == null || fieldKey.isBlank()) {
            errors.reject(path, "FIELD_KEY_EMPTY", "fieldKey must not be empty");
            return null;
        }

        SqlFieldRef existing = fields.get(fieldKey);

        if (existing != null) {
            return existing;
        }

        FieldMetadata field = rootMetadata.findFieldByName(fieldKey)
                .or(() -> rootMetadata.findFieldByJsonName(fieldKey))
                .orElse(null);

        if (field == null) {
            errors.reject(
                    path,
                    "UNKNOWN_FIELD",
                    "Unknown fieldKey '" + fieldKey + "' in class '" + rootMetadata.sourceValue() + "'"
            );
            return null;
        }

        SqlFieldRef ref = new SqlFieldRef(
                fieldKey,
                field.jsonName(),
                field.kind(),
                field
        );

        fields.put(fieldKey, ref);

        return ref;
    }

    private String extractFilterValue(JsonNode condition, String filterType, boolean secondValue) {
        if ("date".equalsIgnoreCase(filterType)) {
            if (secondValue) {
                if (condition.hasNonNull("dateTo")) {
                    return condition.get("dateTo").asText();
                }

                if (condition.hasNonNull("filterTo")) {
                    return condition.get("filterTo").asText();
                }

                return null;
            }

            if (condition.hasNonNull("dateFrom")) {
                return condition.get("dateFrom").asText();
            }
        }

        String fieldName = secondValue ? "filterTo" : "filter";

        if (!condition.hasNonNull(fieldName)) {
            return null;
        }

        return condition.get(fieldName).asText();
    }
}
```

## `ClickHouseDialect.java`

```java
package com.pivotService.SQLBuilder.dialect;

import com.pivotService.metadata.FieldKind;
import org.springframework.stereotype.Component;

import java.util.Arrays;
import java.util.Locale;
import java.util.regex.Pattern;
import java.util.stream.Collectors;

@Component
public class ClickHouseDialect implements SqlDialect {

    private static final Pattern IDENTIFIER_PART =
            Pattern.compile("[A-Za-z_][A-Za-z0-9_]*");

    @Override
    public String quoteIdentifier(String identifier) {
        if (identifier == null || identifier.isBlank()) {
            throw new IllegalArgumentException("Identifier cannot be null or empty");
        }

        return Arrays.stream(identifier.split("\\."))
                .map(this::quoteIdentifierPart)
                .collect(Collectors.joining("."));
    }

    @Override
    public String literal(String rawValue, FieldKind kind) {
        if (rawValue == null) {
            return "NULL";
        }

        String value = unquote(rawValue.trim());

        if ("null".equalsIgnoreCase(value)) {
            return "NULL";
        }

        return switch (kind) {
            case INTEGER, LONG, BIG_DECIMAL -> value;
            case BOOLEAN -> {
                String normalized = value.toLowerCase(Locale.ROOT);
                yield SetBooleanLiteral.of(normalized);
            }
            case LOCAL_DATE -> "toDate(" + stringLiteral(value) + ")";
            case INSTANT -> "parseDateTimeBestEffort(" + stringLiteral(value) + ")";
            case UUID -> "toUUID(" + stringLiteral(value) + ")";
            case STRING -> stringLiteral(value);
        };
    }

    @Override
    public String stringLiteral(String value) {
        if (value == null) {
            return "NULL";
        }

        String escaped = value
                .replace("\\", "\\\\")
                .replace("'", "''");

        return "'" + escaped + "'";
    }

    @Override
    public String dateGroupExpression(String columnExpression, String groupBy, FieldKind sourceKind) {
        if (!sourceKind.isDateLike()) {
            throw new IllegalArgumentException("Date grouping requires date-like field");
        }

        return switch (groupBy.toLowerCase(Locale.ROOT)) {
            case "year" -> "toYear(" + columnExpression + ")";
            case "month" -> "toStartOfMonth(" + columnExpression + ")";
            case "day" -> "toDate(" + columnExpression + ")";
            default -> throw new IllegalArgumentException("Unsupported date groupBy: " + groupBy);
        };
    }

    @Override
    public String dateGroupLiteral(String rawValue, String groupBy) {
        String value = unquote(rawValue.trim());

        return switch (groupBy.toLowerCase(Locale.ROOT)) {
            case "year" -> value;
            case "month", "day" -> "toDate(" + stringLiteral(value) + ")";
            default -> throw new IllegalArgumentException("Unsupported date groupBy: " + groupBy);
        };
    }

    @Override
    public String aggregationExpression(String aggFunc, String columnExpression) {
        String normalized = aggFunc.toLowerCase(Locale.ROOT);

        return switch (normalized) {
            case "unique_count" -> "uniqExact(" + columnExpression + ")";
            case "count" -> "count(" + columnExpression + ")";
            case "sum", "avg", "min", "max" -> normalized + "(" + columnExpression + ")";
            default -> throw new IllegalArgumentException("Unsupported aggregation: " + aggFunc);
        };
    }

    private String quoteIdentifierPart(String part) {
        if (!IDENTIFIER_PART.matcher(part).matches()) {
            throw new IllegalArgumentException("Unsafe SQL identifier: " + part);
        }

        return "`" + part + "`";
    }

    private String unquote(String value) {
        if (value.length() >= 2) {
            boolean singleQuoted = value.startsWith("'") && value.endsWith("'");
            boolean doubleQuoted = value.startsWith("\"") && value.endsWith("\"");

            if (singleQuoted || doubleQuoted) {
                return value.substring(1, value.length() - 1);
            }
        }

        return value;
    }

    private static final class SetBooleanLiteral {

        static String of(String value) {
            return switch (value) {
                case "true", "1" -> "true";
                case "false", "0" -> "false";
                default -> throw new IllegalArgumentException("Invalid boolean literal: " + value);
            };
        }
    }
}
```

# 14. Рефакторинг `StreamViewConfigToSqlConverter`

Ниже — вариант, где SQL-builder уже не валидирует поля, а использует `ValidatedViewConfig`.

Он:

- принимает только валидированный конфиг;
- `fieldKey` из запроса мапит в `jsonName`;
- для SELECT использует SQL-колонку `jsonName`;
- может алиасить результат обратно в `fieldKey`, чтобы frontend получал ожидаемые имена.

```java
package com.pivotService.SQLBuilder;

import com.fasterxml.jackson.databind.JsonNode;
import com.fasterxml.jackson.databind.node.ArrayNode;
import com.fasterxml.jackson.databind.node.JsonNodeFactory;
import com.pivotService.SQLBuilder.dialect.SqlDialect;
import com.pivotService.SQLBuilder.validation.SqlFieldRef;
import com.pivotService.SQLBuilder.validation.ValidatedViewConfig;
import org.springframework.stereotype.Service;

import java.util.*;
import java.util.stream.Collectors;

@Service
public class StreamViewConfigToSqlConverter {

    private final SqlDialect dialect;

    public StreamViewConfigToSqlConverter(SqlDialect dialect) {
        this.dialect = dialect;
    }

    public String convert(ValidatedViewConfig validated) {
        JsonNode config = validated.config();

        ArrayNode columns = getArrayNode(config, "columns");
        ArrayNode rowGroupColumns = getArrayNode(config, "rowGroupColumns");
        ArrayNode pivotColumns = getArrayNode(config, "pivotColumns");
        ArrayNode valueColumns = getArrayNode(config, "valueColumns");
        ArrayNode sortModel = getArrayNode(config, "sortModel");
        ArrayNode groupKeys = getArrayNode(config, "groupKeys");

        JsonNode filterModel = config.path("filterModel");

        int startRow = config.path("startRow").asInt(0);
        int endRow = config.path("endRow").asInt(100);

        columns = ensureIdColumn(columns);

        Map<String, String> aggregationByField = extractAggregationByField(valueColumns);

        boolean hasGrouping = rowGroupColumns != null && !rowGroupColumns.isEmpty();

        ArrayNode cleanGroupColumns = cleanGroupColumns(rowGroupColumns, groupKeys);
        ArrayNode firstRowGroupColumns = firstGroupLevel(cleanGroupColumns, pivotColumns);

        Set<String> actualGroupByFields = extractFieldKeySet(firstRowGroupColumns);

        StringBuilder sql = new StringBuilder();

        sql.append("SELECT ");
        sql.append(buildSelectClause(
                validated,
                columns,
                firstRowGroupColumns,
                valueColumns,
                hasGrouping
        ));

        sql.append(" FROM ");
        sql.append(dialect.quoteIdentifier(validated.tableName()));

        FilterResult filterResult = buildFilters(
                validated,
                filterModel,
                groupKeys,
                rowGroupColumns,
                aggregationByField
        );

        if (!filterResult.whereClause.isEmpty()) {
            sql.append(" WHERE ").append(filterResult.whereClause);
        }

        if (!firstRowGroupColumns.isEmpty()) {
            String groupByClause = buildGroupByClause(validated, firstRowGroupColumns, hasGrouping);

            if (!groupByClause.isEmpty()) {
                sql.append(" GROUP BY ").append(groupByClause);
            }
        }

        if (!filterResult.havingClause.isEmpty()) {
            sql.append(" HAVING ").append(filterResult.havingClause);
        }

        String orderByClause = buildOrderByClause(
                validated,
                sortModel,
                actualGroupByFields,
                valueColumns
        );

        if (!orderByClause.isEmpty()) {
            sql.append(" ORDER BY ").append(orderByClause);
        }

        int limit = endRow - startRow;

        if (limit > 0) {
            sql.append(" LIMIT ").append(limit);
        }

        if (startRow > 0) {
            sql.append(" OFFSET ").append(startRow);
        }

        return sql.toString();
    }

    private String buildSelectClause(
            ValidatedViewConfig validated,
            ArrayNode columns,
            ArrayNode firstRowGroupColumns,
            ArrayNode valueColumns,
            boolean hasGrouping
    ) {
        List<String> selectParts = new ArrayList<>();

        if (!firstRowGroupColumns.isEmpty()) {
            for (JsonNode groupColumn : firstRowGroupColumns) {
                selectParts.add(groupExpression(validated, groupColumn, true));
            }

            addAggregations(validated, selectParts, valueColumns);
            return String.join(", ", selectParts);
        }

        if (hasGrouping) {
            addColumnsToSelect(validated, selectParts, columns);
            return selectParts.isEmpty() ? "*" : String.join(", ", selectParts);
        }

        addColumnsToSelect(validated, selectParts, columns);

        return selectParts.isEmpty() ? "*" : String.join(", ", selectParts);
    }

    private void addColumnsToSelect(
            ValidatedViewConfig validated,
            List<String> selectParts,
            ArrayNode columns
    ) {
        if (columns == null || columns.isEmpty()) {
            selectParts.add("*");
            return;
        }

        for (JsonNode col : columns) {
            String fieldKey = getFieldKey(col);

            if (fieldKey == null) {
                continue;
            }

            SqlFieldRef field = validated.field(fieldKey);

            selectParts.add(
                    columnExpression(field) +
                            " AS " +
                            dialect.quoteIdentifier(field.requestKey())
            );
        }
    }

    private void addAggregations(
            ValidatedViewConfig validated,
            List<String> selectParts,
            ArrayNode valueColumns
    ) {
        if (valueColumns == null) {
            return;
        }

        for (JsonNode valueColumn : valueColumns) {
            String aggFunc = getAggFunc(valueColumn);
            String fieldKey = getFieldKey(valueColumn);

            if (fieldKey == null || aggFunc == null) {
                continue;
            }

            SqlFieldRef field = validated.field(fieldKey);

            String expression = dialect.aggregationExpression(
                    aggFunc,
                    columnExpression(field)
            );

            selectParts.add(expression + " AS " + dialect.quoteIdentifier(field.requestKey()));
        }
    }

    private String buildGroupByClause(
            ValidatedViewConfig validated,
            ArrayNode rowGroupColumns,
            boolean hasGrouping
    ) {
        if (!hasGrouping) {
            return "";
        }

        List<String> groupByParts = new ArrayList<>();

        for (JsonNode node : rowGroupColumns) {
            groupByParts.add(groupExpression(validated, node, false));
        }

        return String.join(", ", groupByParts);
    }

    private String groupExpression(
            ValidatedViewConfig validated,
            JsonNode node,
            boolean addAlias
    ) {
        String fieldKey = node.path("fieldKey").asText(null);

        if (fieldKey == null) {
            throw new IllegalArgumentException("group fieldKey is required");
        }

        SqlFieldRef field = validated.field(fieldKey);

        String expression = columnExpression(field);

        String groupBy = node.path("groupBy").asText(null);

        if (groupBy != null && !groupBy.isBlank()) {
            expression = dialect.dateGroupExpression(
                    expression,
                    groupBy,
                    field.kind()
            );
        }

        if (addAlias) {
            return expression + " AS " + dialect.quoteIdentifier(field.requestKey());
        }

        return expression;
    }

    private FilterResult buildFilters(
            ValidatedViewConfig validated,
            JsonNode filterModel,
            ArrayNode groupKeys,
            ArrayNode rowGroupColumns,
            Map<String, String> aggregationByField
    ) {
        FilterResult result = new FilterResult();

        List<String> whereConditions = new ArrayList<>();
        List<String> havingConditions = new ArrayList<>();

        if (filterModel != null && !filterModel.isMissingNode() && filterModel.isObject()) {
            Iterator<Map.Entry<String, JsonNode>> fields = filterModel.fields();

            while (fields.hasNext()) {
                Map.Entry<String, JsonNode> entry = fields.next();

                String fieldKey = entry.getKey();
                JsonNode filterDef = entry.getValue();

                SqlFieldRef field = validated.field(fieldKey);

                boolean isAggregated = aggregationByField.containsKey(fieldKey);

                String expression;

                if (isAggregated) {
                    expression = dialect.aggregationExpression(
                            aggregationByField.get(fieldKey),
                            columnExpression(field)
                    );
                } else {
                    expression = columnExpression(field);
                }

                String filterSql = buildFilterCondition(field, expression, filterDef);

                if (filterSql != null && !filterSql.isEmpty()) {
                    if (isAggregated) {
                        havingConditions.add(filterSql);
                    } else {
                        whereConditions.add(filterSql);
                    }
                }
            }
        }

        if (groupKeys != null && !groupKeys.isEmpty()) {
            Map<String, String> groupByByField = extractGroupByByField(rowGroupColumns);

            for (JsonNode node : groupKeys) {
                if (!node.has("fieldKey") || !node.has("value")) {
                    continue;
                }

                String fieldKey = node.get("fieldKey").asText();
                String value = node.get("value").asText();

                SqlFieldRef field = validated.field(fieldKey);

                String expression = columnExpression(field);
                String literal;

                String groupBy = node.path("groupBy").asText(null);

                if (groupBy == null || groupBy.isBlank()) {
                    groupBy = groupByByField.get(fieldKey);
                }

                if (groupBy != null && !groupBy.isBlank()) {
                    expression = dialect.dateGroupExpression(
                            expression,
                            groupBy,
                            field.kind()
                    );

                    literal = dialect.dateGroupLiteral(value, groupBy);
                } else {
                    literal = dialect.literal(value, field.kind());
                }

                whereConditions.add(expression + " = " + literal);
            }
        }

        result.whereClause = String.join(" AND ", whereConditions);
        result.havingClause = String.join(" AND ", havingConditions);

        return result;
    }

    private String buildFilterCondition(
            SqlFieldRef field,
            String expression,
            JsonNode filterDef
    ) {
        if (filterDef == null || filterDef.isMissingNode()) {
            return null;
        }

        String filterType = filterDef.path("filterType").asText("");

        if (filterDef.has("operator")) {
            String operator = filterDef.path("operator").asText("AND");

            String cond1 = buildSingleFilterCondition(
                    field,
                    expression,
                    filterDef.get("condition1"),
                    filterType
            );

            String cond2 = buildSingleFilterCondition(
                    field,
                    expression,
                    filterDef.get("condition2"),
                    filterType
            );

            List<String> parts = new ArrayList<>();

            if (cond1 != null) {
                parts.add(cond1);
            }

            if (cond2 != null) {
                parts.add(cond2);
            }

            if (parts.isEmpty()) {
                return null;
            }

            if (parts.size() == 1) {
                return parts.get(0);
            }

            String sqlOperator = "OR".equalsIgnoreCase(operator) ? " OR " : " AND ";

            return "(" + String.join(sqlOperator, parts) + ")";
        }

        if ("set".equalsIgnoreCase(filterType)) {
            return buildSetFilter(field, expression, filterDef);
        }

        return buildSingleFilterCondition(field, expression, filterDef, filterType);
    }

    private String buildSingleFilterCondition(
            SqlFieldRef field,
            String expression,
            JsonNode condition,
            String filterType
    ) {
        if (condition == null || condition.isMissingNode()) {
            return null;
        }

        String type = condition.path("type").asText("");

        return switch (type) {
            case "equals" ->
                    expression + " = " + literal(field, extractFilterValue(condition, filterType, false));

            case "notEqual" ->
                    expression + " != " + literal(field, extractFilterValue(condition, filterType, false));

            case "contains" ->
                    expression + " LIKE " + dialect.stringLiteral("%" + condition.path("filter").asText() + "%");

            case "notContains" ->
                    expression + " NOT LIKE " + dialect.stringLiteral("%" + condition.path("filter").asText() + "%");

            case "startsWith" ->
                    expression + " LIKE " + dialect.stringLiteral(condition.path("filter").asText() + "%");

            case "endsWith" ->
                    expression + " LIKE " + dialect.stringLiteral("%" + condition.path("filter").asText());

            case "lessThan" ->
                    expression + " < " + literal(field, extractFilterValue(condition, filterType, false));

            case "lessThanOrEqual" ->
                    expression + " <= " + literal(field, extractFilterValue(condition, filterType, false));

            case "greaterThan" ->
                    expression + " > " + literal(field, extractFilterValue(condition, filterType, false));

            case "greaterThanOrEqual" ->
                    expression + " >= " + literal(field, extractFilterValue(condition, filterType, false));

            case "inRange" -> {
                String from = extractFilterValue(condition, filterType, false);
                String to = extractFilterValue(condition, filterType, true);

                yield expression + " BETWEEN " + literal(field, from) + " AND " + literal(field, to);
            }

            case "blank", "empty" ->
                    "(" + expression + " IS NULL)";

            case "notBlank", "notEmpty" ->
                    "(" + expression + " IS NOT NULL)";

            default -> {
                if (condition.has("filter")) {
                    yield expression + " = " + literal(field, extractFilterValue(condition, filterType, false));
                }

                yield null;
            }
        };
    }

    private String buildSetFilter(
            SqlFieldRef field,
            String expression,
            JsonNode filterDef
    ) {
        JsonNode values = filterDef.path("values");

        if (!values.isArray() || values.isEmpty()) {
            return expression + " IN (NULL)";
        }

        StringJoiner joiner = new StringJoiner(", ");
        boolean hasNull = false;

        for (JsonNode value : values) {
            if (value.isNull()) {
                hasNull = true;
            } else {
                joiner.add(dialect.literal(value.asText(), field.kind()));
            }
        }

        if (hasNull && joiner.length() > 0) {
            return "(" + expression + " IN (" + joiner + ") OR " + expression + " IS NULL)";
        }

        if (hasNull) {
            return expression + " IS NULL";
        }

        return expression + " IN (" + joiner + ")";
    }

    private String buildOrderByClause(
            ValidatedViewConfig validated,
            ArrayNode sortModel,
            Set<String> actualGroupByFields,
            ArrayNode valueColumns
    ) {
        if (sortModel == null || sortModel.isEmpty()) {
            return "";
        }

        Set<String> valueFields = valueColumns != null
                ? valueColumns.findValuesAsText("fieldKey").stream().collect(Collectors.toSet())
                : Set.of();

        List<String> orderParts = new ArrayList<>();

        for (JsonNode sortItem : sortModel) {
            String colId = sortItem.path("colId").asText(null);
            String sort = sortItem.path("sort").asText("asc");

            if (colId == null || colId.isBlank()) {
                continue;
            }

            if (!isAllowedOrderBy(colId, actualGroupByFields, valueFields)) {
                continue;
            }

            SqlFieldRef field = validated.field(colId);

            String expression;

            if (actualGroupByFields.contains(colId) || valueFields.contains(colId)) {
                expression = dialect.quoteIdentifier(field.requestKey());
            } else {
                expression = columnExpression(field);
            }

            orderParts.add(expression + " " + ("desc".equalsIgnoreCase(sort) ? "DESC" : "ASC"));
        }

        return String.join(", ", orderParts);
    }

    private boolean isAllowedOrderBy(
            String colId,
            Set<String> groupByFields,
            Set<String> valueFields
    ) {
        if (groupByFields.isEmpty() && valueFields.isEmpty()) {
            return true;
        }

        return groupByFields.contains(colId) || valueFields.contains(colId);
    }

    private String columnExpression(SqlFieldRef field) {
        return dialect.quoteIdentifier(field.columnName());
    }

    private String literal(SqlFieldRef field, String value) {
        return dialect.literal(value, field.kind());
    }

    private String extractFilterValue(JsonNode condition, String filterType, boolean secondValue) {
        if ("date".equalsIgnoreCase(filterType)) {
            if (secondValue) {
                if (condition.hasNonNull("dateTo")) {
                    return condition.get("dateTo").asText();
                }

                if (condition.hasNonNull("filterTo")) {
                    return condition.get("filterTo").asText();
                }

                return null;
            }

            if (condition.hasNonNull("dateFrom")) {
                return condition.get("dateFrom").asText();
            }
        }

        String fieldName = secondValue ? "filterTo" : "filter";

        if (!condition.hasNonNull(fieldName)) {
            return null;
        }

        return condition.get(fieldName).asText();
    }

    private Map<String, String> extractAggregationByField(ArrayNode valueColumns) {
        Map<String, String> result = new HashMap<>();

        if (valueColumns == null) {
            return result;
        }

        for (JsonNode valueColumn : valueColumns) {
            String fieldKey = getFieldKey(valueColumn);
            String aggFunc = getAggFunc(valueColumn);

            if (fieldKey != null && aggFunc != null) {
                result.put(fieldKey, aggFunc);
            }
        }

        return result;
    }

    private Map<String, String> extractGroupByByField(ArrayNode rowGroupColumns) {
        Map<String, String> result = new HashMap<>();

        if (rowGroupColumns == null) {
            return result;
        }

        for (JsonNode node : rowGroupColumns) {
            String fieldKey = node.path("fieldKey").asText(null);
            String groupBy = node.path("groupBy").asText(null);

            if (fieldKey != null && groupBy != null && !groupBy.isBlank()) {
                result.put(fieldKey, groupBy);
            }
        }

        return result;
    }

    private ArrayNode ensureIdColumn(ArrayNode columns) {
        ArrayNode result = columns;

        if (result == null) {
            result = JsonNodeFactory.instance.arrayNode();
        }

        boolean hasId = false;

        for (JsonNode col : result) {
            if ("id".equals(col.path("fieldKey").asText(null))) {
                hasId = true;
                break;
            }
        }

        if (!hasId) {
            var idCol = JsonNodeFactory.instance.objectNode();
            idCol.put("fieldKey", "id");
            result.add(idCol);
        }

        return result;
    }

    private ArrayNode cleanGroupColumns(ArrayNode rowGroupColumns, ArrayNode groupKeys) {
        ArrayNode result = JsonNodeFactory.instance.arrayNode();

        if (rowGroupColumns == null) {
            return result;
        }

        Set<String> rowGroupFieldKeys = extractFieldKeySet(rowGroupColumns);
        Set<String> groupKeySet = extractFieldKeySet(groupKeys);

        rowGroupFieldKeys.removeAll(groupKeySet);

        for (JsonNode column : rowGroupColumns) {
            String fieldKey = column.path("fieldKey").asText(null);

            if (fieldKey != null && rowGroupFieldKeys.contains(fieldKey)) {
                result.add(column);
            }
        }

        return result;
    }

    private ArrayNode firstGroupLevel(ArrayNode cleanGroupColumns, ArrayNode pivotColumns) {
        ArrayNode result = JsonNodeFactory.instance.arrayNode();

        if (cleanGroupColumns != null && !cleanGroupColumns.isEmpty()) {
            result.add(cleanGroupColumns.get(0));
        }

        if (pivotColumns != null && !pivotColumns.isEmpty()) {
            result.addAll(pivotColumns);
        }

        return result;
    }

    private Set<String> extractFieldKeySet(ArrayNode arrayNode) {
        Set<String> result = new HashSet<>();

        if (arrayNode == null) {
            return result;
        }

        for (JsonNode node : arrayNode) {
            JsonNode field = node.get("fieldKey");

            if (field != null) {
                result.add(field.asText());
            }
        }

        return result;
    }

    private String getFieldKey(JsonNode column) {
        return column != null ? column.path("fieldKey").asText(null) : null;
    }

    private String getAggFunc(JsonNode column) {
        if (column == null) {
            return null;
        }

        String aggFunc = column.path("aggFunc").asText(null);

        if (aggFunc == null) {
            return null;
        }

        return switch (aggFunc.toLowerCase(Locale.ROOT)) {
            case "sum", "avg", "min", "max", "count", "unique_count" -> aggFunc.toLowerCase(Locale.ROOT);
            default -> null;
        };
    }

    private ArrayNode getArrayNode(JsonNode parent, String fieldName) {
        JsonNode node = parent.path(fieldName);
        return node.isArray() ? (ArrayNode) node : null;
    }

    private static class FilterResult {
        String whereClause = "";
        String havingClause = "";
    }
}
```

# 15. Что важно изменить в текущей логике

## 1. Не использовать `sanitizeIdentifier` как “очистку”

Ваш текущий метод:

```java
identifier.replaceAll("[^a-zA-Z0-9_.]", "")
```

опасен тем, что он молча меняет имя.

Например:

```text
abc;DROP TABLE x
```

превратится в:

```text
abcDROPTABLEx
```

Лучше:

- сначала проверить поле через metadata registry;
- затем экранировать известное имя колонки из метаданных;
- если идентификатор невалидный — выбросить ошибку.

## 2. SQL-builder не должен валидировать бизнес-смысл

Правильное разделение:

```text
ViewConfigValidator
    проверяет поля, типы, restrictions, aggregation, filters

StreamViewConfigToSqlConverter
    строит SQL только из валидированной модели
```

## 3. Для SQL используйте `jsonName`, но можно возвращать alias как `name`

Если в запросе пришло:

```json
{
  "fieldKey": "base_amount"
}
```

а в metadata:

```json
{
  "name": "base_amount",
  "jsonName": "baseAmount"
}
```

SQL лучше строить так:

```sql
SELECT `baseAmount` AS `base_amount`
```

Так frontend продолжит получать ключ `base_amount`, но база будет читать правильную колонку `baseAmount`.

## Рекомендуемый порядок внедрения

1. Сначала добавьте DTO и `BusinessMetadataService`.
2. Подключите обновление registry из вашего существующего контроллера метаданных.
3. Добавьте `BusinessPathResolver`.
4. Добавьте `LiveBookRequestValidator`.
5. Добавьте `ViewConfigValidator`.
6. После этого замените старый `SqlQueryBuilder` на версию через `ValidatedViewConfig`.
7. Потом постепенно переносите SQL-логику в маленькие классы: `SelectBuilder`, `FilterBuilder`, `GroupByBuilder`, `OrderByBuilder`.

Главная польза такого подхода: вся бизнес-валидация и работа с типами будет сосредоточена до SQL-builder, а SQL-builder станет значительно проще и безопаснее.
