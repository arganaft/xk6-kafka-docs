Да, для `v2/config` лучше **вообще не использовать глобальный поиск поля по `fieldKey`**. Именно он и порождает ошибки вида:

```text
Ambiguous fieldKey 'instrument_id'
```

В `v2` у вас теперь есть полное поле:

```json
{
  "fieldKey": "FxSpotForwardTrade.baseCurrency.code"
}
```

Это уже однозначный путь. Поэтому новая логика должна быть такой:

```text
fieldKey из v2/config
  -> проверить, что этот путь есть в fields выбранного LiveBook
  -> через resolver получить физическое имя колонки ClickHouse
  -> построить SQL по физическим колонкам
```

Например:

```text
FxSpotForwardTrade.baseAmount
  -> baseAmount

FxSpotForwardTrade.baseCurrency.code
  -> baseCurrency_code

Trade.portfolioId
  -> portfolioId

Entity.id
  -> id
```

# 1. Что должно получиться

Если frontend отправит в `v2/config`:

```json
{
  "type": "SetViewConfiguration",
  "data": {
    "liveBookName": "LiveBook250",
    "startRow": 0,
    "endRow": 1000,
    "columns": [
      {
        "fieldKey": "Entity.id"
      },
      {
        "fieldKey": "FxSpotForwardTrade.baseAmount"
      },
      {
        "fieldKey": "FxSpotForwardTrade.baseCurrency.code"
      },
      {
        "fieldKey": "Trade.portfolioId"
      }
    ],
    "rowGroupColumns": [],
    "valueColumns": [],
    "pivotColumns": [],
    "groupKeys": [],
    "filterModel": {},
    "sortModel": []
  }
}
```

SQL должен быть примерно таким:

```sql
SELECT
  "id" AS "Entity.id",
  "baseAmount" AS "FxSpotForwardTrade.baseAmount",
  "baseCurrency_code" AS "FxSpotForwardTrade.baseCurrency.code",
  "portfolioId" AS "Trade.portfolioId"
FROM "default"."livebook250"
LIMIT 1000
```

Главное:

- `"baseCurrency_code"` — реальная колонка в ClickHouse;
- `"FxSpotForwardTrade.baseCurrency.code"` — alias для frontend;
- глобальный `fieldKey = instrument_id` больше не используется;
- коллизии между классами больше не мешают.

# 3. Добавьте roots в `BusinessMetadataService`

Новый resolver должен работать не через глобальную карту полей, а по дереву классов, потому что нужно повторить логику приложения, которое создает таблицы.

Измените `BusinessMetadataService`.

```java
package com.pivotService.metadata;

import com.fasterxml.jackson.core.type.TypeReference;
import com.fasterxml.jackson.databind.JsonNode;
import com.fasterxml.jackson.databind.ObjectMapper;
import com.pivotService.metadata.dto.BusinessClassNode;
import org.springframework.stereotype.Service;

import java.util.List;
import java.util.concurrent.atomic.AtomicReference;

@Service
public class BusinessMetadataService {

    private final ObjectMapper objectMapper;

    private final AtomicReference<BusinessMetadataRegistry> registryRef =
            new AtomicReference<>(BusinessMetadataRegistry.empty());

    private final AtomicReference<List<BusinessClassNode>> rootsRef =
            new AtomicReference<>(List.of());

    public BusinessMetadataService(ObjectMapper objectMapper) {
        this.objectMapper = objectMapper;
    }

    public void refresh(JsonNode hierarchyJson) {
        try {
            List<BusinessClassNode> roots;

            if (hierarchyJson.isArray()) {
                roots = objectMapper.convertValue(
                        hierarchyJson,
                        new TypeReference<List<BusinessClassNode>>() {
                        }
                );
            } else {
                roots = List.of(objectMapper.treeToValue(hierarchyJson, BusinessClassNode.class));
            }

            rootsRef.set(List.copyOf(roots));
            registryRef.set(BusinessMetadataRegistry.from(roots));
        } catch (Exception e) {
            throw new IllegalArgumentException("Cannot parse business metadata hierarchy", e);
        }
    }

    public BusinessMetadataRegistry registry() {
        BusinessMetadataRegistry registry = registryRef.get();

        if (registry.isEmpty()) {
            throw new IllegalStateException("Business metadata registry is empty");
        }

        return registry;
    }

    public List<BusinessClassNode> roots() {
        List<BusinessClassNode> roots = rootsRef.get();

        if (roots.isEmpty()) {
            throw new IllegalStateException("Business metadata roots are empty");
        }

        return roots;
    }
}
```

# 6. Resolver пути LiveBook field -> ClickHouse column

Это аналог вашей логики из приложения, которое создает ClickHouse-таблицы.

## `V2DataDictionaryPathResolver.java`

```java
package com.pivotService.SQLBuilder.v2.resolver;

import com.pivotService.SQLBuilder.v2.livebook.V2ResolvedLiveBookField;
import com.pivotService.metadata.FieldKind;
import com.pivotService.metadata.FieldMetadata;
import com.pivotService.metadata.dto.BusinessClassNode;
import com.pivotService.metadata.dto.BusinessField;
import com.pivotService.metadata.dto.BusinessRelation;

import java.util.*;
import java.util.stream.Collectors;

public class V2DataDictionaryPathResolver {

    private record FieldInfo(
            String declaredInClass,
            BusinessField field
    ) {
        String jsonName() {
            return field.effectiveJsonName();
        }

        String type() {
            return field.type();
        }
    }

    private final Map<String, BusinessClassNode> bySourceValue = new HashMap<>();
    private final Map<BusinessClassNode, BusinessClassNode> parentOf = new IdentityHashMap<>();

    public V2DataDictionaryPathResolver(List<BusinessClassNode> roots) {
        for (BusinessClassNode root : roots) {
            index(root, null);
        }
    }

    public V2ResolvedLiveBookField resolveScalar(String path) {
        List<V2ResolvedLiveBookField> resolved = resolve(path);

        if (resolved.size() != 1) {
            throw new IllegalArgumentException(
                    "Path '" + path + "' resolved to " + resolved.size() +
                            " columns. v2/config currently supports only scalar field paths."
            );
        }

        return resolved.get(0);
    }

    public List<V2ResolvedLiveBookField> resolve(String path) {
        if (path == null || path.isBlank()) {
            throw new IllegalArgumentException("Empty LiveBook field path");
        }

        String[] tokens = path.split("\\.");

        if (tokens.length < 2) {
            throw new IllegalArgumentException("Path must have format Class.field or Class.relation.field: " + path);
        }

        BusinessClassNode current = bySourceValue.get(tokens[0]);

        if (current == null) {
            throw new NoSuchElementException(
                    "No entity with sourceValue='" + tokens[0] + "' for path '" + path + "'"
            );
        }

        List<List<String>> segments = new ArrayList<>();
        List<String> segment = new ArrayList<>();

        BusinessClassNode wholeSetTarget = null;
        FieldInfo leaf = null;

        for (int i = 1; i < tokens.length; i++) {
            String token = tokens[i];
            boolean last = i == tokens.length - 1;

            if (last) {
                FieldInfo field = findFieldInherited(current, token);

                if (field != null) {
                    segment.add(field.jsonName());
                    segments.add(segment);
                    leaf = field;
                    break;
                }

                BusinessRelation embeddedSet = findRelationInherited(current, token);

                if (embeddedSet != null && isEmbeddedSet(embeddedSet)) {
                    segment.add(token);
                    segments.add(segment);

                    BusinessClassNode target = bySourceValue.get(embeddedSet.targetClass());

                    if (target == null) {
                        throw new NoSuchElementException(
                                "No targetClass='" + embeddedSet.targetClass() + "' for path '" + path + "'"
                        );
                    }

                    wholeSetTarget = target;
                    break;
                }

                throw new NoSuchElementException(
                        "Field or embedded_set '" + token + "' was not found in '" +
                                current.sourceValue() + "' for path '" + path + "'"
                );
            }

            BusinessRelation relation = findRelationInherited(current, token);

            if (relation == null) {
                throw new NoSuchElementException(
                        "Relation '" + token + "' was not found in '" +
                                current.sourceValue() + "' for path '" + path + "'"
                );
            }

            segment.add(token);

            if (isEmbeddedSet(relation)) {
                segments.add(segment);
                segment = new ArrayList<>();
            }

            BusinessClassNode next = bySourceValue.get(relation.targetClass());

            if (next == null) {
                throw new NoSuchElementException(
                        "No targetClass='" + relation.targetClass() +
                                "' for relation '" + token + "' in path '" + path + "'"
                );
            }

            current = next;
        }

        if (wholeSetTarget == null) {
            if (leaf == null) {
                throw new IllegalArgumentException("Cannot resolve leaf field for path '" + path + "'");
            }

            return List.of(toSingleResolvedField(path, segments, leaf));
        }

        return resolveTerminalEmbeddedSet(path, segments, wholeSetTarget);
    }

    private V2ResolvedLiveBookField toSingleResolvedField(
            String originalPath,
            List<List<String>> segments,
            FieldInfo leaf
    ) {
        if (segments.size() == 1) {
            List<String> jsonPath = segments.get(0);
            String columnName = String.join("_", jsonPath);

            return toResolvedField(
                    originalPath,
                    columnName,
                    leaf,
                    jsonPath,
                    false
            );
        }

        List<String> firstArray = segments.get(0);
        List<List<String>> inner = new ArrayList<>(segments.subList(1, segments.size()));

        String group = String.join("_", firstArray);
        String subName = inner.stream()
                .flatMap(List::stream)
                .collect(Collectors.joining("_"));

        String columnName = group + "." + subName;

        List<String> flatJsonPath = segments.stream()
                .flatMap(List::stream)
                .toList();

        return toResolvedField(
                originalPath,
                columnName,
                leaf,
                flatJsonPath,
                true
        );
    }

    private List<V2ResolvedLiveBookField> resolveTerminalEmbeddedSet(
            String originalPath,
            List<List<String>> segments,
            BusinessClassNode wholeSetTarget
    ) {
        List<String> firstArray = segments.get(0);
        List<List<String>> midInner = new ArrayList<>(segments.subList(1, segments.size()));

        String group = String.join("_", firstArray);

        List<V2ResolvedLiveBookField> result = new ArrayList<>();

        for (FieldInfo fieldInfo : collectInheritedFields(wholeSetTarget)) {
            List<List<String>> inner = new ArrayList<>(midInner);
            inner.add(List.of(fieldInfo.jsonName()));

            String subName = inner.stream()
                    .flatMap(List::stream)
                    .collect(Collectors.joining("_"));

            String columnName = group + "." + subName;

            String syntheticPath = originalPath + "." + fieldInfo.jsonName();

            List<String> flatJsonPath = inner.stream()
                    .flatMap(List::stream)
                    .toList();

            result.add(toResolvedField(
                    syntheticPath,
                    columnName,
                    fieldInfo,
                    flatJsonPath,
                    true
            ));
        }

        if (result.isEmpty()) {
            throw new NoSuchElementException(
                    "Target class '" + wholeSetTarget.sourceValue() +
                            "' has no fields for path '" + originalPath + "'"
            );
        }

        return result;
    }

    private V2ResolvedLiveBookField toResolvedField(
            String fieldPath,
            String columnName,
            FieldInfo fieldInfo,
            List<String> jsonPath,
            boolean nested
    ) {
        FieldKind kind = FieldKind.from(fieldInfo.type());

        FieldMetadata metadata = new FieldMetadata(
                fieldInfo.declaredInClass(),
                fieldInfo.field(),
                kind
        );

        return new V2ResolvedLiveBookField(
                fieldPath,
                columnName,
                fieldPath,
                fieldInfo.type(),
                kind,
                metadata,
                List.copyOf(jsonPath),
                nested
        );
    }

    private void index(BusinessClassNode entity, BusinessClassNode parent) {
        if (entity == null) {
            return;
        }

        if (hasText(entity.sourceValue())) {
            bySourceValue.putIfAbsent(entity.sourceValue(), entity);
        }

        if (parent != null) {
            parentOf.put(entity, parent);
        }

        for (BusinessClassNode child : entity.children()) {
            index(child, entity);
        }
    }

    private FieldInfo findFieldInherited(BusinessClassNode entity, String jsonName) {
        for (BusinessClassNode current = entity; current != null; current = parentOf.get(current)) {
            for (BusinessField field : current.fields()) {
                if (jsonName.equals(field.effectiveJsonName())) {
                    String declaredInClass = hasText(current.sourceValue())
                            ? current.sourceValue()
                            : current.name();

                    return new FieldInfo(declaredInClass, field);
                }
            }
        }

        return null;
    }

    private List<FieldInfo> collectInheritedFields(BusinessClassNode entity) {
        LinkedHashMap<String, FieldInfo> result = new LinkedHashMap<>();

        for (BusinessClassNode current = entity; current != null; current = parentOf.get(current)) {
            String declaredInClass = hasText(current.sourceValue())
                    ? current.sourceValue()
                    : current.name();

            for (BusinessField field : current.fields()) {
                result.putIfAbsent(
                        field.effectiveJsonName(),
                        new FieldInfo(declaredInClass, field)
                );
            }
        }

        return new ArrayList<>(result.values());
    }

    private BusinessRelation findRelationInherited(BusinessClassNode entity, String token) {
        for (BusinessClassNode current = entity; current != null; current = parentOf.get(current)) {
            for (BusinessRelation relation : current.relations()) {
                if (relationMatches(relation, token)) {
                    return relation;
                }
            }
        }

        return null;
    }

    private boolean relationMatches(BusinessRelation relation, String token) {
        if (isEmbeddedSet(relation)) {
            return token.equals(relation.name());
        }

        if (hasText(relation.alias())) {
            return token.equals(relation.alias());
        }

        return token.equals(relation.name());
    }

    private boolean isEmbeddedSet(BusinessRelation relation) {
        return "embedded_set".equalsIgnoreCase(relation.type());
    }

    private static boolean hasText(String value) {
        return value != null && !value.isBlank();
    }
}
```

## `V2LiveBookTableNameResolver.java`

У вас таблица создается как:

хотя LiveBook называется:

```java
package com.pivotService.SQLBuilder.v2.livebook;

import com.fasterxml.jackson.databind.JsonNode;
import org.springframework.beans.factory.annotation.Value;
import org.springframework.stereotype.Component;

import java.util.Locale;

@Component
public class V2LiveBookTableNameResolver {

    @Value("${app.clickhouse.database:default}")
    private String database;

    @Value("${app.v2.livebook-table-lowercase:true}")
    private boolean lowercaseTableName;

    public String resolve(JsonNode config, String liveBookName) {
        if (config.hasNonNull("tableName")) {
            return config.get("tableName").asText();
        }

        String table = liveBookName;

        if (lowercaseTableName) {
            table = table.toLowerCase(Locale.ROOT);
        }

        if (database == null || database.isBlank()) {
            return table;
        }

        return database + "." + table;
    }
}
```

## `V2LiveBookSchemaService.java`

```java
package com.pivotService.SQLBuilder.v2.livebook;

import com.fasterxml.jackson.databind.JsonNode;
import com.pivotService.SQLBuilder.v2.resolver.V2DataDictionaryPathResolver;
import com.pivotService.livebook.dto.LiveBookDto;
import com.pivotService.livebook.service.LiveBookService;
import com.pivotService.metadata.BusinessMetadataService;
import org.springframework.stereotype.Service;

import java.util.LinkedHashMap;
import java.util.List;
import java.util.Map;

@Service
public class V2LiveBookSchemaService {

    private final LiveBookService liveBookService;
    private final BusinessMetadataService businessMetadataService;
    private final V2LiveBookTableNameResolver tableNameResolver;

    public V2LiveBookSchemaService(
            LiveBookService liveBookService,
            BusinessMetadataService businessMetadataService,
            V2LiveBookTableNameResolver tableNameResolver
    ) {
        this.liveBookService = liveBookService;
        this.businessMetadataService = businessMetadataService;
        this.tableNameResolver = tableNameResolver;
    }

    public V2LiveBookSchema resolveSchema(JsonNode config) {
        LiveBookDto liveBook = resolveLiveBook(config);
        String tableName = tableNameResolver.resolve(config, liveBook.liveBookName());

        return buildSchema(liveBook, tableName);
    }

    private LiveBookDto resolveLiveBook(JsonNode config) {
        List<LiveBookDto> liveBooks = liveBookService.findAll();

        if (config.hasNonNull("liveBookId")) {
            long liveBookId = config.get("liveBookId").asLong();

            return liveBooks.stream()
                    .filter(liveBook -> liveBook.liveBookId() != null)
                    .filter(liveBook -> liveBook.liveBookId() == liveBookId)
                    .findFirst()
                    .orElseThrow(() -> new IllegalArgumentException(
                            "LiveBook with id=" + liveBookId + " was not found"
                    ));
        }

        if (config.hasNonNull("liveBookName")) {
            String liveBookName = config.get("liveBookName").asText();

            return liveBooks.stream()
                    .filter(liveBook -> liveBookName.equals(liveBook.liveBookName()))
                    .findFirst()
                    .orElseThrow(() -> new IllegalArgumentException(
                            "LiveBook with name='" + liveBookName + "' was not found"
                    ));
        }

        throw new IllegalArgumentException(
                "v2/config requires liveBookId or liveBookName in data"
        );
    }

    private V2LiveBookSchema buildSchema(LiveBookDto liveBook, String tableName) {
        V2DataDictionaryPathResolver resolver =
                new V2DataDictionaryPathResolver(businessMetadataService.roots());

        Map<String, V2ResolvedLiveBookField> fieldsByPath = new LinkedHashMap<>();

        for (String fieldPath : liveBook.fields()) {
            try {
                V2ResolvedLiveBookField resolved = resolver.resolveScalar(fieldPath);
                fieldsByPath.put(fieldPath, resolved);
            } catch (Exception e) {
                throw new IllegalArgumentException(
                        "Cannot resolve LiveBook field '" + fieldPath +
                                "' for liveBook '" + liveBook.liveBookName() + "'",
                        e
                );
            }
        }

        return new V2LiveBookSchema(
                liveBook.liveBookId(),
                liveBook.liveBookName(),
                tableName,
                fieldsByPath
        );
    }
}
```

# 9. ClickHouse dialect для v2

Важно: для alias вида:

```text
FxSpotForwardTrade.baseCurrency.code
```

нельзя делать так:

```sql
"FxSpotForwardTrade"."baseCurrency"."code"
```

Нужно quotes вокруг всего alias:

```sql
"FxSpotForwardTrade.baseCurrency.code"
```

## `V2ClickHouseDialect.java`

```java
package com.pivotService.SQLBuilder.v2.dialect;

import com.pivotService.metadata.FieldKind;
import org.springframework.stereotype.Component;

import java.util.Arrays;
import java.util.Locale;
import java.util.regex.Pattern;
import java.util.stream.Collectors;

@Component
public class V2ClickHouseDialect {

    private static final Pattern SAFE_TABLE_PART =
            Pattern.compile("[A-Za-z_][A-Za-z0-9_]*");

    private static final Pattern SAFE_COLUMN_OR_ALIAS =
            Pattern.compile("[A-Za-z_][A-Za-z0-9_\\.]*");

    public String quoteTable(String tableName) {
        if (tableName == null || tableName.isBlank()) {
            throw new IllegalArgumentException("Table name cannot be empty");
        }

        return Arrays.stream(tableName.split("\\."))
                .map(this::quoteTablePart)
                .collect(Collectors.joining("."));
    }

    public String quoteColumn(String columnName) {
        return quoteWholeIdentifier(columnName);
    }

    public String quoteAlias(String alias) {
        return quoteWholeIdentifier(alias);
    }

    public String literal(String rawValue, FieldKind kind) {
        if (rawValue == null) {
            return "NULL";
        }

        String value = unquote(rawValue.trim());

        if ("null".equalsIgnoreCase(value)) {
            return "NULL";
        }

        return switch (kind) {
            case INTEGER, LONG, BIG_DECIMAL -> value;
            case BOOLEAN -> booleanLiteral(value);
            case LOCAL_DATE -> "toDate(" + stringLiteral(value) + ")";
            case INSTANT -> "parseDateTimeBestEffort(" + stringLiteral(value) + ")";
            case UUID -> "toUUID(" + stringLiteral(value) + ")";
            case STRING -> stringLiteral(value);
        };
    }

    public String stringLiteral(String value) {
        if (value == null) {
            return "NULL";
        }

        String escaped = value
                .replace("\\", "\\\\")
                .replace("'", "''");

        return "'" + escaped + "'";
    }

    public String aggregationExpression(String aggFunc, String columnExpression) {
        String normalized = aggFunc.toLowerCase(Locale.ROOT);

        return switch (normalized) {
            case "unique_count" -> "uniqExact(" + columnExpression + ")";
            case "count" -> "count(" + columnExpression + ")";
            case "sum", "avg", "min", "max" -> normalized + "(" + columnExpression + ")";
            default -> throw new IllegalArgumentException("Unsupported aggregation: " + aggFunc);
        };
    }

    public String dateGroupExpression(
            String columnExpression,
            String groupBy,
            FieldKind kind
    ) {
        if (!kind.isDateLike()) {
            throw new IllegalArgumentException("Date grouping requires date-like field");
        }

        return switch (groupBy.toLowerCase(Locale.ROOT)) {
            case "year" -> "toYear(" + columnExpression + ")";
            case "month" -> "toStartOfMonth(" + columnExpression + ")";
            case "day" -> "toDate(" + columnExpression + ")";
            default -> throw new IllegalArgumentException("Unsupported date groupBy: " + groupBy);
        };
    }

    public String dateGroupLiteral(String rawValue, String groupBy) {
        String value = unquote(rawValue.trim());

        return switch (groupBy.toLowerCase(Locale.ROOT)) {
            case "year" -> value;
            case "month", "day" -> "toDate(" + stringLiteral(value) + ")";
            default -> throw new IllegalArgumentException("Unsupported date groupBy: " + groupBy);
        };
    }

    private String quoteTablePart(String part) {
        if (!SAFE_TABLE_PART.matcher(part).matches()) {
            throw new IllegalArgumentException("Unsafe table identifier: " + part);
        }

        return "\"" + part + "\"";
    }

    private String quoteWholeIdentifier(String identifier) {
        if (identifier == null || identifier.isBlank()) {
            throw new IllegalArgumentException("Identifier cannot be empty");
        }

        if (!SAFE_COLUMN_OR_ALIAS.matcher(identifier).matches()) {
            throw new IllegalArgumentException("Unsafe SQL identifier: " + identifier);
        }

        return "\"" + identifier.replace("\"", "\"\"") + "\"";
    }

    private String booleanLiteral(String value) {
        String normalized = value.toLowerCase(Locale.ROOT);

        return switch (normalized) {
            case "true", "1" -> "true";
            case "false", "0" -> "false";
            default -> throw new IllegalArgumentException("Invalid boolean literal: " + value);
        };
    }

    private String unquote(String value) {
        if (value.length() >= 2) {
            boolean singleQuoted = value.startsWith("'") && value.endsWith("'");
            boolean doubleQuoted = value.startsWith("\"") && value.endsWith("\"");

            if (singleQuoted || doubleQuoted) {
                return value.substring(1, value.length() - 1);
            }
        }

        return value;
    }
}
```

# 10. Валидатор v2/config

Главная идея: `fieldKey` должен быть не просто известным полем, а полем из конкретного LiveBook.

## `V2ViewConfigValidator.java`

```java
package com.pivotService.SQLBuilder.v2.validation;

import com.fasterxml.jackson.databind.JsonNode;
import com.pivotService.SQLBuilder.v2.livebook.V2LiveBookSchema;
import com.pivotService.SQLBuilder.v2.livebook.V2LiveBookSchemaService;
import com.pivotService.SQLBuilder.v2.livebook.V2ResolvedLiveBookField;
import com.pivotService.metadata.FieldKind;
import com.pivotService.metadata.FieldMetadata;
import com.pivotService.metadata.dto.BusinessField;
import com.pivotService.validation.FieldValueValidator;
import com.pivotService.validation.ValidationErrors;
import org.springframework.stereotype.Service;

import java.util.LinkedHashMap;
import java.util.List;
import java.util.Locale;
import java.util.Map;
import java.util.Set;

@Service
public class V2ViewConfigValidator {

    private static final Set<String> AGG_FUNCS =
            Set.of("sum", "avg", "min", "max", "count", "unique_count");

    private static final Set<String> SORT_DIRECTIONS =
            Set.of("asc", "desc");

    private final V2LiveBookSchemaService schemaService;
    private final FieldValueValidator fieldValueValidator;

    public V2ViewConfigValidator(
            V2LiveBookSchemaService schemaService,
            FieldValueValidator fieldValueValidator
    ) {
        this.schemaService = schemaService;
        this.fieldValueValidator = fieldValueValidator;
    }

    public V2ValidatedViewConfig validate(JsonNode requestOrConfig) {
        JsonNode config = unwrapData(requestOrConfig);

        ValidationErrors errors = new ValidationErrors();

        if (config == null || !config.isObject()) {
            errors.reject("$", "INVALID_CONFIG", "v2 config data must be JSON object");
            errors.throwIfAny();
        }

        V2LiveBookSchema schema;

        try {
            schema = schemaService.resolveSchema(config);
        } catch (Exception e) {
            errors.reject("$.liveBook", "LIVEBOOK_SCHEMA_ERROR", e.getMessage());
            errors.throwIfAny();
            return null;
        }

        Map<String, V2SqlFieldRef> fields = new LinkedHashMap<>();

        validatePaging(config, errors);

        validateFieldArray(config, "columns", schema, fields, errors);
        validateFieldArray(config, "rowGroupColumns", schema, fields, errors);
        validateFieldArray(config, "pivotColumns", schema, fields, errors);

        validateValueColumns(config, schema, fields, errors);
        validateSortModel(config, schema, fields, errors);
        validateGroupKeys(config, schema, fields, errors);
        validateFilterModel(config.path("filterModel"), schema, fields, errors);

        addTechnicalIdField(fields, schema);

        errors.throwIfAny();

        return new V2ValidatedViewConfig(
                config,
                schema,
                fields
        );
    }

    private JsonNode unwrapData(JsonNode requestOrConfig) {
        if (requestOrConfig != null
                && requestOrConfig.has("data")
                && requestOrConfig.get("data").isObject()) {
            return requestOrConfig.get("data");
        }

        return requestOrConfig;
    }

    private void validatePaging(JsonNode config, ValidationErrors errors) {
        int startRow = config.path("startRow").asInt(0);
        int endRow = config.path("endRow").asInt(100);

        if (startRow < 0) {
            errors.reject("$.startRow", "INVALID_START_ROW", "startRow must be >= 0");
        }

        if (endRow < startRow) {
            errors.reject("$.endRow", "INVALID_END_ROW", "endRow must be >= startRow");
        }
    }

    private void validateFieldArray(
            JsonNode config,
            String arrayName,
            V2LiveBookSchema schema,
            Map<String, V2SqlFieldRef> fields,
            ValidationErrors errors
    ) {
        JsonNode array = config.path(arrayName);

        if (array.isMissingNode() || array.isNull()) {
            return;
        }

        if (!array.isArray()) {
            errors.reject("$." + arrayName, "INVALID_ARRAY", arrayName + " must be array");
            return;
        }

        for (int i = 0; i < array.size(); i++) {
            JsonNode item = array.get(i);
            String fieldKey = item.path("fieldKey").asText(null);

            V2SqlFieldRef field = resolveField(
                    schema,
                    fieldKey,
                    "$." + arrayName + "[" + i + "].fieldKey",
                    fields,
                    errors
            );

            validateGroupByIfPresent(
                    item,
                    field,
                    "$." + arrayName + "[" + i + "]",
                    errors
            );
        }
    }

    private void validateValueColumns(
            JsonNode config,
            V2LiveBookSchema schema,
            Map<String, V2SqlFieldRef> fields,
            ValidationErrors errors
    ) {
        JsonNode valueColumns = config.path("valueColumns");

        if (valueColumns.isMissingNode() || valueColumns.isNull()) {
            return;
        }

        if (!valueColumns.isArray()) {
            errors.reject("$.valueColumns", "INVALID_ARRAY", "valueColumns must be array");
            return;
        }

        for (int i = 0; i < valueColumns.size(); i++) {
            JsonNode item = valueColumns.get(i);

            String fieldKey = item.path("fieldKey").asText(null);
            String aggFunc = item.path("aggFunc").asText(null);

            V2SqlFieldRef field = resolveField(
                    schema,
                    fieldKey,
                    "$.valueColumns[" + i + "].fieldKey",
                    fields,
                    errors
            );

            validateAggFunc(
                    field,
                    aggFunc,
                    "$.valueColumns[" + i + "].aggFunc",
                    errors
            );
        }
    }

    private void validateSortModel(
            JsonNode config,
            V2LiveBookSchema schema,
            Map<String, V2SqlFieldRef> fields,
            ValidationErrors errors
    ) {
        JsonNode sortModel = config.path("sortModel");

        if (sortModel.isMissingNode() || sortModel.isNull()) {
            return;
        }

        if (!sortModel.isArray()) {
            errors.reject("$.sortModel", "INVALID_ARRAY", "sortModel must be array");
            return;
        }

        for (int i = 0; i < sortModel.size(); i++) {
            JsonNode item = sortModel.get(i);

            String colId = item.path("colId").asText(null);
            String sort = item.path("sort").asText("asc").toLowerCase(Locale.ROOT);

            resolveField(
                    schema,
                    colId,
                    "$.sortModel[" + i + "].colId",
                    fields,
                    errors
            );

            if (!SORT_DIRECTIONS.contains(sort)) {
                errors.reject(
                        "$.sortModel[" + i + "].sort",
                        "INVALID_SORT_DIRECTION",
                        "sort must be one of " + SORT_DIRECTIONS
                );
            }
        }
    }

    private void validateGroupKeys(
            JsonNode config,
            V2LiveBookSchema schema,
            Map<String, V2SqlFieldRef> fields,
            ValidationErrors errors
    ) {
        JsonNode groupKeys = config.path("groupKeys");

        if (groupKeys.isMissingNode() || groupKeys.isNull()) {
            return;
        }

        if (!groupKeys.isArray()) {
            errors.reject("$.groupKeys", "INVALID_ARRAY", "groupKeys must be array");
            return;
        }

        for (int i = 0; i < groupKeys.size(); i++) {
            JsonNode item = groupKeys.get(i);

            String fieldKey = item.path("fieldKey").asText(null);

            V2SqlFieldRef field = resolveField(
                    schema,
                    fieldKey,
                    "$.groupKeys[" + i + "].fieldKey",
                    fields,
                    errors
            );

            if (field != null && item.has("value")) {
                fieldValueValidator.validateLiteral(
                        field.metadata(),
                        item.get("value").asText(),
                        "$.groupKeys[" + i + "].value",
                        errors
                );
            }
        }
    }

    private void validateFilterModel(
            JsonNode filterModel,
            V2LiveBookSchema schema,
            Map<String, V2SqlFieldRef> fields,
            ValidationErrors errors
    ) {
        if (filterModel == null || filterModel.isMissingNode() || filterModel.isNull()) {
            return;
        }

        if (!filterModel.isObject()) {
            errors.reject("$.filterModel", "INVALID_OBJECT", "filterModel must be object");
            return;
        }

        filterModel.fields().forEachRemaining(entry -> {
            String fieldKey = entry.getKey();

            V2SqlFieldRef field = resolveField(
                    schema,
                    fieldKey,
                    "$.filterModel." + fieldKey,
                    fields,
                    errors
            );

            if (field != null) {
                validateFilterValueIfPossible(
                        field,
                        entry.getValue(),
                        "$.filterModel." + fieldKey,
                        errors
                );
            }
        });
    }

    private void validateFilterValueIfPossible(
            V2SqlFieldRef field,
            JsonNode filterDef,
            String path,
            ValidationErrors errors
    ) {
        if (filterDef == null || filterDef.isMissingNode() || filterDef.isNull()) {
            return;
        }

        if (filterDef.has("operator")) {
            validateFilterValueIfPossible(field, filterDef.path("condition1"), path + ".condition1", errors);
            validateFilterValueIfPossible(field, filterDef.path("condition2"), path + ".condition2", errors);
            return;
        }

        String type = filterDef.path("type").asText("");
        String filterType = filterDef.path("filterType").asText("");

        if ("set".equalsIgnoreCase(filterType)) {
            JsonNode values = filterDef.path("values");

            if (values.isArray()) {
                for (int i = 0; i < values.size(); i++) {
                    if (!values.get(i).isNull()) {
                        fieldValueValidator.validateLiteral(
                                field.metadata(),
                                values.get(i).asText(),
                                path + ".values[" + i + "]",
                                errors
                        );
                    }
                }
            }

            return;
        }

        switch (type) {
            case "equals", "notEqual", "lessThan", "lessThanOrEqual",
                 "greaterThan", "greaterThanOrEqual" -> {
                String value = extractFilterValue(filterDef, filterType, false);

                if (value != null) {
                    fieldValueValidator.validateLiteral(field.metadata(), value, path, errors);
                }
            }

            case "inRange" -> {
                String from = extractFilterValue(filterDef, filterType, false);
                String to = extractFilterValue(filterDef, filterType, true);

                if (from != null) {
                    fieldValueValidator.validateLiteral(field.metadata(), from, path + ".from", errors);
                }

                if (to != null) {
                    fieldValueValidator.validateLiteral(field.metadata(), to, path + ".to", errors);
                }
            }

            case "contains", "notContains", "startsWith", "endsWith" -> {
                if (!field.kind().isTextLike()) {
                    errors.reject(
                            path + ".type",
                            "INVALID_FILTER_OPERATION",
                            "Text filter operation is not allowed for field type " + field.kind()
                    );
                }

                String value = filterDef.path("filter").asText(null);

                if (value != null) {
                    fieldValueValidator.validateLiteral(field.metadata(), value, path + ".filter", errors);
                }
            }

            case "blank", "notBlank", "empty", "notEmpty", "" -> {
                // value not required
            }

            default -> errors.reject(
                    path + ".type",
                    "UNKNOWN_FILTER_OPERATION",
                    "Unknown filter operation: " + type
            );
        }
    }

    private String extractFilterValue(JsonNode condition, String filterType, boolean secondValue) {
        if ("date".equalsIgnoreCase(filterType)) {
            if (secondValue) {
                if (condition.hasNonNull("dateTo")) {
                    return condition.get("dateTo").asText();
                }

                if (condition.hasNonNull("filterTo")) {
                    return condition.get("filterTo").asText();
                }

                return null;
            }

            if (condition.hasNonNull("dateFrom")) {
                return condition.get("dateFrom").asText();
            }
        }

        String fieldName = secondValue ? "filterTo" : "filter";

        if (!condition.hasNonNull(fieldName)) {
            return null;
        }

        return condition.get(fieldName).asText();
    }

    private V2SqlFieldRef resolveField(
            V2LiveBookSchema schema,
            String fieldKey,
            String path,
            Map<String, V2SqlFieldRef> fields,
            ValidationErrors errors
    ) {
        if (fieldKey == null || fieldKey.isBlank()) {
            errors.reject(path, "FIELD_KEY_EMPTY", "fieldKey must not be empty");
            return null;
        }

        V2SqlFieldRef existing = fields.get(fieldKey);

        if (existing != null) {
            return existing;
        }

        if ("id".equals(fieldKey)) {
            V2SqlFieldRef idRef = technicalIdRef();
            fields.put(fieldKey, idRef);
            return idRef;
        }

        V2ResolvedLiveBookField liveBookField = schema.findField(fieldKey).orElse(null);

        if (liveBookField == null) {
            errors.reject(
                    path,
                    "UNKNOWN_LIVEBOOK_FIELD",
                    "Unknown fieldKey '" + fieldKey + "' for LiveBook '" +
                            schema.liveBookName() + "'. v2/config expects full field path from LiveBook.fields."
            );
            return null;
        }

        V2SqlFieldRef ref = new V2SqlFieldRef(
                fieldKey,
                liveBookField.columnName(),
                liveBookField.alias(),
                liveBookField.kind(),
                liveBookField.fieldMetadata()
        );

        fields.put(fieldKey, ref);

        return ref;
    }

    private void addTechnicalIdField(
            Map<String, V2SqlFieldRef> fields,
            V2LiveBookSchema schema
    ) {
        boolean alreadyHasIdColumn = fields.values()
                .stream()
                .anyMatch(field -> "id".equals(field.columnName()));

        if (alreadyHasIdColumn) {
            return;
        }

        fields.put("id", technicalIdRef());
    }

    private V2SqlFieldRef technicalIdRef() {
        BusinessField idField = new BusinessField(
                "id",
                "LONG",
                "id",
                List.of()
        );

        FieldMetadata metadata = new FieldMetadata(
                "Entity",
                idField,
                FieldKind.LONG
        );

        return new V2SqlFieldRef(
                "id",
                "id",
                "id",
                FieldKind.LONG,
                metadata
        );
    }

    private void validateGroupByIfPresent(
            JsonNode item,
            V2SqlFieldRef field,
            String path,
            ValidationErrors errors
    ) {
        if (field == null || !item.hasNonNull("groupBy")) {
            return;
        }

        String groupBy = item.get("groupBy").asText().toLowerCase(Locale.ROOT);

        if (!Set.of("year", "month", "day").contains(groupBy)) {
            errors.reject(
                    path + ".groupBy",
                    "INVALID_GROUP_BY",
                    "groupBy must be one of year, month, day"
            );
            return;
        }

        if (!field.kind().isDateLike()) {
            errors.reject(
                    path + ".groupBy",
                    "GROUP_BY_REQUIRES_DATE",
                    "Date groupBy requires date-like field, but got " + field.kind()
            );
        }
    }

    private void validateAggFunc(
            V2SqlFieldRef field,
            String aggFunc,
            String path,
            ValidationErrors errors
    ) {
        if (aggFunc == null || aggFunc.isBlank()) {
            errors.reject(path, "AGG_FUNC_REQUIRED", "aggFunc is required");
            return;
        }

        String normalized = aggFunc.toLowerCase(Locale.ROOT);

        if (!AGG_FUNCS.contains(normalized)) {
            errors.reject(path, "INVALID_AGG_FUNC", "aggFunc must be one of " + AGG_FUNCS);
            return;
        }

        if (field == null) {
            return;
        }

        if (Set.of("sum", "avg").contains(normalized) && !field.kind().isNumeric()) {
            errors.reject(
                    path,
                    "AGG_FUNC_REQUIRES_NUMERIC",
                    "Aggregation '" + normalized + "' requires numeric field, but got " + field.kind()
            );
        }
    }
}
```

# 12. Как должен отличаться v2 converter от старого

Можно взять ваш текущий `StreamViewConfigToSqlConverter`, скопировать его в:

```text
com.pivotService.SQLBuilder.v2.V2StreamViewConfigToSqlConverter
```

И заменить три ключевых места.

---

## 12.1. Колонка в SQL

Было в старом API:

```java
sanitizeIdentifier(fieldKey)
```

В v2 должно быть:

```java
private String columnExpression(V2SqlFieldRef field) {
    return dialect.quoteColumn(field.columnName());
}
```

Например:

```text
fieldKey = FxSpotForwardTrade.baseCurrency.code
columnName = baseCurrency_code
```

SQL:

```sql
"baseCurrency_code"
```

```java
private String aliasExpression(V2SqlFieldRef field) {
```
