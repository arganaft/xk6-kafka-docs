Да, можно сделать **интеграционный тест**, который:

1. поднимает Spring-контекст;
2. берет ваш уже настроенный `RestTemplate`;
3. вызывает `app.metaDataTreeURL`;
4. обновляет `BusinessMetadataService`;
5. строит `BusinessMetadataRegistry`;
6. проходит по всем полям;
7. ищет коллизии;
8. логирует каждую коллизию в консоль;
9. падает, если коллизии найдены.

```java
package com.pivotService.metadata;

import com.fasterxml.jackson.core.type.TypeReference;
import com.fasterxml.jackson.databind.ObjectMapper;
import com.pivotService.metadata.dto.BusinessClassNode;
import org.junit.jupiter.api.Test;

import java.util.List;

import static org.junit.jupiter.api.Assertions.*;

class BusinessMetadataRegistryTest {

    private final ObjectMapper objectMapper = new ObjectMapper();

    @Test
    void shouldResolveCompatibleDuplicateGlobalField() throws Exception {
        BusinessMetadataRegistry registry = registryFromJson("""
                [
                  {
                    "name": "TRADE",
                    "sourceValue": "Trade",
                    "fields": [
                      {
                        "name": "side",
                        "type": "STRING",
                        "jsonName": "side",
                        "restrictions": []
                      }
                    ],
                    "relations": [],
                    "children": []
                  },
                  {
                    "name": "TRADE_SAVE_EVENT",
                    "sourceValue": "TradeSaveEvent",
                    "fields": [
                      {
                        "name": "side",
                        "type": "STRING",
                        "jsonName": "side",
                        "restrictions": []
                      }
                    ],
                    "relations": [],
                    "children": []
                  }
                ]
                """);

        assertDoesNotThrow(() -> registry.findGlobalField("side"));

        FieldMetadata field = registry.findGlobalField("side").orElseThrow();

        assertEquals("side", field.name());
        assertEquals("side", field.jsonName());
        assertEquals(FieldKind.STRING, field.kind());
    }

    @Test
    void shouldThrowAmbiguousFieldWhenSameNameHasDifferentTypes() throws Exception {
        BusinessMetadataRegistry registry = registryFromJson("""
                [
                  {
                    "name": "FIXING_SOURCE",
                    "sourceValue": "FixingSource",
                    "fields": [
                      {
                        "name": "instrument_id",
                        "type": "STRING",
                        "jsonName": "instrumentId",
                        "restrictions": []
                      }
                    ],
                    "relations": [],
                    "children": []
                  },
                  {
                    "name": "TRADE",
                    "sourceValue": "Trade",
                    "fields": [
                      {
                        "name": "instrument_id",
                        "type": "LONG",
                        "jsonName": "instrumentId",
                        "restrictions": []
                      }
                    ],
                    "relations": [],
                    "children": []
                  }
                ]
                """);

        IllegalArgumentException exception = assertThrows(
                IllegalArgumentException.class,
                () -> registry.findGlobalField("instrument_id")
        );

        assertTrue(exception.getMessage().contains("Ambiguous fieldKey 'instrument_id'"));
        assertTrue(exception.getMessage().contains("FixingSource.instrument_id"));
        assertTrue(exception.getMessage().contains("Trade.instrument_id"));
    }

    @Test
    void childShouldInheritParentFields() throws Exception {
        BusinessMetadataRegistry registry = registryFromJson("""
                [
                  {
                    "name": "ENTITY",
                    "sourceValue": "Entity",
                    "fields": [
                      {
                        "name": "id",
                        "type": "LONG",
                        "jsonName": "id",
                        "restrictions": []
                      }
                    ],
                    "relations": [],
                    "children": [
                      {
                        "name": "TRADE",
                        "sourceValue": "Trade",
                        "fields": [
                          {
                            "name": "status",
                            "type": "STRING",
                            "jsonName": "status",
                            "restrictions": []
                          }
                        ],
                        "relations": [],
                        "children": []
                      }
                    ]
                  }
                ]
                """);

        BusinessClassMetadata trade = registry.requireClass("Trade");

        assertTrue(trade.findFieldByName("id").isPresent());
        assertTrue(trade.findFieldByJsonName("id").isPresent());

        assertTrue(trade.findFieldByName("status").isPresent());
        assertTrue(trade.findFieldByJsonName("status").isPresent());
    }

    private BusinessMetadataRegistry registryFromJson(String json) throws Exception {
        List<BusinessClassNode> roots = objectMapper.readValue(
                json,
                new TypeReference<List<BusinessClassNode>>() {
                }
        );

        return BusinessMetadataRegistry.from(roots);
    }
}
```

# 2. Интеграционный тест с реальным `RestTemplate`

Файл:

```text
src/test/java/com/pivotService/metadata/BusinessMetadataRegistryRealMetadataTest.java
```

```java
package com.pivotService.metadata;

import com.fasterxml.jackson.databind.JsonNode;
import org.junit.jupiter.api.Tag;
import org.junit.jupiter.api.Test;
import org.slf4j.Logger;
import org.slf4j.LoggerFactory;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.beans.factory.annotation.Value;
import org.springframework.boot.test.context.SpringBootTest;
import org.springframework.web.client.RestTemplate;

import java.util.*;
import java.util.function.Function;
import java.util.stream.Collectors;

import static org.junit.jupiter.api.Assertions.*;

@Tag("integration")
@SpringBootTest(webEnvironment = SpringBootTest.WebEnvironment.NONE)
class BusinessMetadataRegistryRealMetadataTest {

    private static final Logger log =
            LoggerFactory.getLogger(BusinessMetadataRegistryRealMetadataTest.class);

    @Autowired
    private RestTemplate restTemplate;

    @Autowired
    private BusinessMetadataService businessMetadataService;

    @Value("${app.metaDataTreeURL}")
    private String metaDataTreeURL;

    @Test
    void realMetadataShouldNotHaveAmbiguousGlobalFieldNames() {
        JsonNode metaDataTree = restTemplate.getForObject(metaDataTreeURL, JsonNode.class);

        assertNotNull(metaDataTree, "metaDataTree response must not be null");

        businessMetadataService.refresh(metaDataTree);

        BusinessMetadataRegistry registry = businessMetadataService.registry();

        Collection<FieldMetadata> allFields = registry.allGlobalFields();

        assertFalse(allFields.isEmpty(), "Registry should contain fields");

        List<FieldCollision> collisionsByName = findCollisions(
                allFields,
                "field.name",
                FieldMetadata::name
        );

        logCollisions("COLLISIONS BY field.name", collisionsByName);

        if (!collisionsByName.isEmpty()) {
            fail(buildFailureMessage(collisionsByName));
        }
    }

    @Test
    void realMetadataShouldNotHaveAmbiguousGlobalJsonNames() {
        JsonNode metaDataTree = restTemplate.getForObject(metaDataTreeURL, JsonNode.class);

        assertNotNull(metaDataTree, "metaDataTree response must not be null");

        businessMetadataService.refresh(metaDataTree);

        BusinessMetadataRegistry registry = businessMetadataService.registry();

        Collection<FieldMetadata> allFields = registry.allGlobalFields();

        assertFalse(allFields.isEmpty(), "Registry should contain fields");

        List<FieldCollision> collisionsByJsonName = findCollisions(
                allFields,
                "field.jsonName",
                FieldMetadata::jsonName
        );

        logCollisions("COLLISIONS BY field.jsonName", collisionsByJsonName);

        if (!collisionsByJsonName.isEmpty()) {
            fail(buildFailureMessage(collisionsByJsonName));
        }
    }

    @Test
    void realMetadataFindGlobalFieldShouldResolveEveryKnownFieldName() {
        JsonNode metaDataTree = restTemplate.getForObject(metaDataTreeURL, JsonNode.class);

        assertNotNull(metaDataTree, "metaDataTree response must not be null");

        businessMetadataService.refresh(metaDataTree);

        BusinessMetadataRegistry registry = businessMetadataService.registry();

        Set<String> fieldNames = registry.allGlobalFields()
                .stream()
                .map(FieldMetadata::name)
                .filter(BusinessMetadataRegistryRealMetadataTest::hasText)
                .collect(Collectors.toCollection(TreeSet::new));

        List<String> errors = new ArrayList<>();

        for (String fieldName : fieldNames) {
            try {
                registry.findGlobalField(fieldName);
            } catch (IllegalArgumentException e) {
                errors.add(e.getMessage());
            }
        }

        for (String error : errors) {
            log.error("findGlobalField error: {}", error);
        }

        if (!errors.isEmpty()) {
            fail("Found " + errors.size() + " fieldKeys that cannot be resolved. See logs.");
        }
    }

    private static List<FieldCollision> findCollisions(
            Collection<FieldMetadata> fields,
            String indexName,
            Function<FieldMetadata, String> keyExtractor
    ) {
        Map<String, List<FieldMetadata>> fieldsByKey = fields.stream()
                .filter(field -> hasText(keyExtractor.apply(field)))
                .collect(Collectors.groupingBy(
                        keyExtractor,
                        TreeMap::new,
                        Collectors.toList()
                ));

        List<FieldCollision> result = new ArrayList<>();

        for (Map.Entry<String, List<FieldMetadata>> entry : fieldsByKey.entrySet()) {
            Optional<FieldCollision> collision = toCollision(
                    indexName,
                    entry.getKey(),
                    entry.getValue()
            );

            collision.ifPresent(result::add);
        }

        return result;
    }

    private static Optional<FieldCollision> toCollision(
            String indexName,
            String fieldKey,
            List<FieldMetadata> candidates
    ) {
        if (candidates.size() <= 1) {
            return Optional.empty();
        }

        Map<FieldSignature, List<FieldMetadata>> bySignature = candidates.stream()
                .collect(Collectors.groupingBy(
                        FieldSignature::from,
                        LinkedHashMap::new,
                        Collectors.toList()
                ));

        if (bySignature.size() <= 1) {
            return Optional.empty();
        }

        return Optional.of(new FieldCollision(
                indexName,
                fieldKey,
                candidates,
                bySignature
        ));
    }

    private static void logCollisions(
            String title,
            List<FieldCollision> collisions
    ) {
        if (collisions.isEmpty()) {
            log.info("{}: no collisions found", title);
            return;
        }

        log.error("{}: found {} collisions", title, collisions.size());

        for (FieldCollision collision : collisions) {
            log.error(
                    "AMBIGUOUS {} '{}': {} different signatures, {} candidates",
                    collision.indexName(),
                    collision.fieldKey(),
                    collision.groups().size(),
                    collision.candidates().size()
            );

            for (Map.Entry<FieldSignature, List<FieldMetadata>> group : collision.groups().entrySet()) {
                FieldSignature signature = group.getKey();

                log.error(
                        "  signature: name='{}', jsonName='{}', kind={}",
                        signature.name(),
                        signature.jsonName(),
                        signature.kind()
                );

                List<String> descriptions = group.getValue()
                        .stream()
                        .map(BusinessMetadataRegistryRealMetadataTest::describeField)
                        .sorted()
                        .toList();

                for (String description : descriptions) {
                    log.error("    - {}", description);
                }
            }
        }
    }

    private static String buildFailureMessage(List<FieldCollision> collisions) {
        String summary = collisions.stream()
                .limit(20)
                .map(collision -> collision.indexName() +
                        " '" +
                        collision.fieldKey() +
                        "' signatures=" +
                        collision.groups().keySet())
                .collect(Collectors.joining(System.lineSeparator()));

        return "Found " + collisions.size() + " ambiguous global field keys. " +
                "See logs for full details." +
                System.lineSeparator() +
                summary;
    }

    private static String describeField(FieldMetadata field) {
        return field.declaredInClass() +
                "." +
                field.name() +
                " jsonName=" +
                field.jsonName() +
                " kind=" +
                field.kind() +
                " rawType=" +
                field.rawType() +
                " restrictions=" +
                field.field().restrictions();
    }

    private static boolean hasText(String value) {
        return value != null && !value.isBlank();
    }

    private record FieldSignature(
            String name,
            String jsonName,
            FieldKind kind
    ) {
        static FieldSignature from(FieldMetadata field) {
            return new FieldSignature(
                    field.name(),
                    field.jsonName(),
                    field.kind()
            );
        }
    }

    private record FieldCollision(
            String indexName,
            String fieldKey,
            List<FieldMetadata> candidates,
            Map<FieldSignature, List<FieldMetadata>> groups
    ) {
    }
}
```

# 3. Что покажет тест для вашего случая

Для такой ошибки:

```text
Ambiguous fieldKey 'instrument_id'. Found different field definitions:
FixingSource.instrument_id jsonName=instrumentId type=STRING;
Trade.instrument_id jsonName=instrumentId type=LONG;
TradeSaveEvent.instrument_id jsonName=instrumentId type=LONG;
TradeFixingEvent.instrument_id jsonName=instrumentId type=LONG.
```

в логах будет примерно так:

```text
AMBIGUOUS field.name 'instrument_id': 2 different signatures, 4 candidates

  signature: name='instrument_id', jsonName='instrumentId', kind=STRING
    - FixingSource.instrument_id jsonName=instrumentId kind=STRING rawType=STRING restrictions=[]

  signature: name='instrument_id', jsonName='instrumentId', kind=LONG
    - Trade.instrument_id jsonName=instrumentId kind=LONG rawType=LONG restrictions=[]
    - TradeFixingEvent.instrument_id jsonName=instrumentId kind=LONG rawType=LONG restrictions=[]
    - TradeSaveEvent.instrument_id jsonName=instrumentId kind=LONG rawType=LONG restrictions=[]
```

Это значит, что коллизия реальная: одно и то же поле `instrument_id` указывает на одну SQL-колонку `instrumentId`, но типы разные: `STRING` и `LONG`.

# 4. Важный момент про `MetaModelClient`

У вас есть:

```java
@PostConstruct
public void init() {
    JsonNode metaDataTree = restTemplate.getForObject(metaDataTreeURL, JsonNode.class);
    businessMetadataService.refresh(metaDataTree);
}
```

Поэтому при запуске `@SpringBootTest` этот код уже вызовет `RestTemplate`.

Но в тесте выше я специально еще раз вызываю:

```java
JsonNode metaDataTree = restTemplate.getForObject(metaDataTreeURL, JsonNode.class);
businessMetadataService.refresh(metaDataTree);
```

Чтобы тест явно проверял актуальные данные из текущего URL.

Если не хотите двойной вызов, можно заменить начало теста на:

```java
BusinessMetadataRegistry registry = businessMetadataService.registry();
```

# 5. Если нужно только логировать, но не падать

Сейчас тест падает при коллизиях:

```java
fail(buildFailureMessage(collisionsByName));
```

Если вы хотите только печатать коллизии в консоль, но чтобы тест проходил, замените:

```java
if (!collisionsByName.isEmpty()) {
    fail(buildFailureMessage(collisionsByName));
}
```

на:

```java
if (!collisionsByName.isEmpty()) {
    log.warn(buildFailureMessage(collisionsByName));
}
```

Но я бы рекомендовал оставлять `fail`, потому что такие коллизии потом приводят к ошибкам валидации `AMBIGUOUS_FIELD`.

# 7. Что делать с найденными коллизиями

Для `instrument_id` есть несколько вариантов:

1. **Исправить metadata**, если `FixingSource.instrument_id` на самом деле тоже должен быть `LONG`.
2. **Исключить некоторые классы из глобального SQL-индекса**, если они не могут попадать в LiveBook.
3. **Перейти к LiveBook-specific schema**, то есть валидировать `SetViewConfiguration` не по всем полям всех классов, а по конкретному набору полей, который был выбран при создании LiveBook.

Третий вариант архитектурно самый правильный, но для диагностики текущий тест уже покажет все проблемные поля.
