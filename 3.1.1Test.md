Ниже — шаблон **контрактного WebSocket-теста**: поднимаем реальный Spring WebSocket/STOMP endpoint, но **ClickHouse не поднимаем**, а `ClickHouseService` заменяем Mockito-моком и возвращаем заранее собранный `QueryData`.

Это не совсем unit-тест. Это лучше назвать **integration/contract test with mocked DB boundary**:

- реальный WebSocket/STOMP слой проверяется;
- реальная сериализация JSON проверяется;
- реальная цепочка обработки запроса проверяется;
- база данных заменена мокнутым `ClickHouseService`.

> Важно: в твоих примерах `request.liveBookId = 298`, а в ответе `config.liveBookId = 354`. Для теста я сделал консистентный вариант с `298`. Если у тебя бизнес-логика действительно преобразует `298 -> 354`, просто поменяй fixture.

## 4. WebSocket contract test

Главный тест проверяет:

1. frontend отправляет `SetViewConfiguration` на `/app/config`;
2. backend сначала отправляет `ConfigurationUpdated`;
3. затем backend отправляет `ReceiveData`;
4. формат JSON совпадает с contract fixture;
5. значения совпадают;
6. `timestamp` есть и имеет ISO-формат;
7. `totalCount` не попал внутрь строк.

> Единственное место, которое нельзя вывести из показанного тобой кода, — destination, куда backend отправляет ответы.  
> Замени `DASHBOARD_MESSAGES_DESTINATION` на тот destination, на который реально подписан frontend: например `/user/queue/trades`, `/user/queue/config`, `/topic/trades` и т.д.

### `src/test/java/com/pivotService/controller/TradeDashboardWebSocketContractTest.java`

```java
package com.pivotService.controller;

import com.fasterxml.jackson.databind.DeserializationFeature;
import com.fasterxml.jackson.databind.JsonNode;
import com.fasterxml.jackson.databind.ObjectMapper;
import com.fasterxml.jackson.databind.node.JsonNodeFactory;
import com.pivotService.resultBroker.clickHouse.ClickHouseService;
import com.pivotService.test.JsonTestResources;
import org.junit.jupiter.api.AfterEach;
import org.junit.jupiter.api.BeforeEach;
import org.junit.jupiter.api.DisplayName;
import org.junit.jupiter.api.Test;
import org.springframework.boot.test.context.SpringBootTest;
import org.springframework.boot.test.web.server.LocalServerPort;
import org.springframework.messaging.converter.MappingJackson2MessageConverter;
import org.springframework.messaging.simp.stomp.StompCommand;
import org.springframework.messaging.simp.stomp.StompFrameHandler;
import org.springframework.messaging.simp.stomp.StompHeaders;
import org.springframework.messaging.simp.stomp.StompSession;
import org.springframework.messaging.simp.stomp.StompSessionHandlerAdapter;
import org.springframework.test.context.bean.override.mockito.MockitoBean;
import org.springframework.util.MimeTypeUtils;
import org.springframework.web.socket.client.standard.StandardWebSocketClient;
import org.springframework.web.socket.messaging.WebSocketStompClient;

import java.lang.reflect.Type;
import java.util.List;
import java.util.concurrent.BlockingQueue;
import java.util.concurrent.CopyOnWriteArrayList;
import java.util.concurrent.LinkedBlockingQueue;

import static com.pivotService.test.assertions.PivotAssertions.assertThatJson;
import static java.util.concurrent.TimeUnit.SECONDS;
import static org.assertj.core.api.BDDAssertions.then;
import static org.mockito.ArgumentMatchers.anyString;
import static org.mockito.BDDMockito.given;
import static org.mockito.Mockito.atLeastOnce;

@SpringBootTest(
        webEnvironment = SpringBootTest.WebEnvironment.RANDOM_PORT,
        properties = {
                "spring.flyway.enabled=false",
                "spring.liquibase.enabled=false"
        }
)
@DisplayName("WebSocket contract: dashboard data flow")
class TradeDashboardWebSocketContractTest {

    private static final String CONFIG_REQUEST_DESTINATION = "/app/config";

    /**
     * Замени на destination, на который реально подписывается frontend.
     *
     * Примеры:
     * - /user/queue/trades
     * - /user/queue/config
     * - /topic/trades
     */
    private static final String DASHBOARD_MESSAGES_DESTINATION = "/user/queue/trades";

    private static final String REQUEST_ID = "227[noGroup][][noPivot][noSort]{noFilter}[0-3]";
    private static final String CLIENT_ID = "d4abe313-8dbf-29dc-5663-fc5a501bee39";

    @LocalServerPort
    private int port;

    /**
     * Spring Boot 3.4+.
     *
     * Если у тебя Spring Boot <= 3.3, замени на:
     *
     * @MockBean
     * private ClickHouseService clickHouseService;
     */
    @MockitoBean
    private ClickHouseService clickHouseService;

    private final ObjectMapper applicationObjectMapper;

    private ObjectMapper contractObjectMapper;
    private WebSocketStompClient stompClient;
    private StompSession stompSession;
    private BlockingQueue<JsonNode> receivedFrames;
    private List<Throwable> asyncErrors;

    TradeDashboardWebSocketContractTest(ObjectMapper applicationObjectMapper) {
        this.applicationObjectMapper = applicationObjectMapper;
    }

    @BeforeEach
    void connectToWebSocket() throws Exception {
        contractObjectMapper = applicationObjectMapper.copy()
                .enable(DeserializationFeature.USE_BIG_DECIMAL_FOR_FLOATS);

        /*
         * Важно для проверки формата чисел.
         * Так тест сможет отличить 250000000.000000 от 250000000.0.
         */
        contractObjectMapper.setNodeFactory(JsonNodeFactory.withExactBigDecimals(true));

        receivedFrames = new LinkedBlockingQueue<>();
        asyncErrors = new CopyOnWriteArrayList<>();

        MappingJackson2MessageConverter messageConverter = new MappingJackson2MessageConverter();
        messageConverter.setObjectMapper(contractObjectMapper);

        stompClient = new WebSocketStompClient(new StandardWebSocketClient());
        stompClient.setMessageConverter(messageConverter);
        stompClient.setInboundMessageSizeLimit(2 * 1024 * 1024);

        stompSession = stompClient
                .connectAsync(
                        "ws://localhost:%d/ws".formatted(port),
                        new RecordingStompSessionHandler()
                )
                .get(3, SECONDS);

        stompSession.subscribe(DASHBOARD_MESSAGES_DESTINATION, new JsonFrameHandler());
    }

    @AfterEach
    void disconnectFromWebSocket() {
        if (stompSession != null && stompSession.isConnected()) {
            stompSession.disconnect();
        }

        if (stompClient != null) {
            stompClient.stop();
        }
    }

    @Test
    @DisplayName("SetViewConfiguration: server sends ConfigurationUpdated and then initial ReceiveData")
    void setViewConfiguration_sendsConfigurationUpdatedAndInitialReceiveData() throws Exception {
        given(clickHouseService.executeAndExtract(anyString()))
                .willReturn(QueryDataFixtures.initialRowsForDashboard());

        sendJson(CONFIG_REQUEST_DESTINATION, "set-view-configuration.json");

        JsonNode configurationUpdated = nextFrame("ConfigurationUpdated");
        JsonNode initialData = nextFrame("initial ReceiveData");

        assertConfigurationUpdated(configurationUpdated);
        assertReceiveData(initialData, "receive-data.initial.expected.json");

        then(asyncErrors)
                .as("STOMP async errors")
                .isEmpty();

        org.mockito.BDDMockito.then(clickHouseService)
                .should(atLeastOnce())
                .executeAndExtract(anyString());
    }

    private void sendJson(String destination, String resourceName) {
        JsonNode request = JsonTestResources.readJson(contractObjectMapper, getClass(), resourceName);

        StompHeaders headers = new StompHeaders();
        headers.setDestination(destination);
        headers.setContentType(MimeTypeUtils.APPLICATION_JSON);

        stompSession.send(headers, request);
    }

    private JsonNode nextFrame(String expectedFrameDescription) throws InterruptedException {
        then(asyncErrors)
                .as("STOMP async errors before waiting for %s", expectedFrameDescription)
                .isEmpty();

        JsonNode frame = receivedFrames.poll(5, SECONDS);

        then(frame)
                .withFailMessage(
                        "Did not receive STOMP MESSAGE <%s> within timeout. Async errors: %s",
                        expectedFrameDescription,
                        asyncErrors
                )
                .isNotNull();

        return frame;
    }

    private void assertConfigurationUpdated(JsonNode frame) {
        assertThatJson(frame)
                .hasOnlyTopLevelFields(
                        "success",
                        "config",
                        "timestamp",
                        "errorMessage",
                        "type"
                )
                .hasTextAt("/type", "ConfigurationUpdated")
                .hasBooleanAt("/success", true)
                .hasObjectAt("/config")
                .hasIntegerAt("/config/liveBookId", 298)
                .hasTextAt("/config/requestId", REQUEST_ID)
                .hasTextAt("/config/clientId", CLIENT_ID)
                .hasTextAt("/config/liveBookName", "livebook298")
                .hasNullAt("/errorMessage")
                .hasIsoInstantAt("/timestamp")
                .isEqualToJsonResourceIgnoring(
                        contractObjectMapper,
                        getClass(),
                        "configuration-updated.expected.json",
                        "/timestamp"
                );
    }

    private void assertReceiveData(JsonNode frame, String expectedResourceName) {
        assertThatJson(frame)
                .hasOnlyTopLevelFields(
                        "type",
                        "success",
                        "data"
                )
                .hasTextAt("/type", "ReceiveData")
                .hasBooleanAt("/success", true)
                .hasArrayAt("/data")
                .hasTextAt("/data/0/requestId", REQUEST_ID)
                .hasIntegerAt("/data/0/totalCount", 57475)
                .hasArraySizeAt("/data/0/rows", 3)

                /*
                 * Если SQL возвращает totalCount как служебную колонку,
                 * наружу она должна попасть только как data[0].totalCount,
                 * но не внутрь каждой row.
                 */
                .doesNotHavePath("/data/0/rows/0/totalCount")
                .doesNotHavePath("/data/0/rows/1/totalCount")
                .doesNotHavePath("/data/0/rows/2/totalCount")

                .isEqualToJsonResource(
                        contractObjectMapper,
                        getClass(),
                        expectedResourceName
                );
    }

    private final class JsonFrameHandler implements StompFrameHandler {

        @Override
        public Type getPayloadType(StompHeaders headers) {
            return JsonNode.class;
        }

        @Override
        public void handleFrame(StompHeaders headers, Object payload) {
            if (payload instanceof JsonNode jsonNode) {
                receivedFrames.add(jsonNode);
                return;
            }

            asyncErrors.add(new AssertionError(
                    "Expected STOMP payload to be JsonNode, but got: " +
                            (payload == null ? "null" : payload.getClass().getName())
            ));
        }
    }

    private final class RecordingStompSessionHandler extends StompSessionHandlerAdapter {

        @Override
        public void handleException(
                StompSession session,
                StompCommand command,
                StompHeaders headers,
                byte[] payload,
                Throwable exception
        ) {
            asyncErrors.add(exception);
        }

        @Override
        public void handleTransportError(StompSession session, Throwable exception) {
            asyncErrors.add(exception);
        }
    }
}
```

```java
package com.pivotService.test;

import com.fasterxml.jackson.databind.JsonNode;
import com.fasterxml.jackson.databind.ObjectMapper;

import java.io.IOException;
import java.io.InputStream;
import java.io.UncheckedIOException;
import java.nio.charset.StandardCharsets;

public final class JsonTestResources {

    private JsonTestResources() {
    }

    public static String readString(Class<?> anchorClass, String resourceName) {
        String relativePath = anchorClass.getSimpleName() + "/" + resourceName;

        try (InputStream inputStream = anchorClass.getResourceAsStream(relativePath)) {
            if (inputStream == null) {
                throw new IllegalArgumentException(
                        "Test resource not found: " +
                                anchorClass.getPackageName().replace('.', '/') +
                                "/" +
                                relativePath
                );
            }

            return new String(inputStream.readAllBytes(), StandardCharsets.UTF_8);
        } catch (IOException exception) {
            throw new UncheckedIOException("Cannot read test resource: " + relativePath, exception);
        }
    }

    public static JsonNode readJson(
            ObjectMapper objectMapper,
            Class<?> anchorClass,
            String resourceName
    ) {
        String json = readString(anchorClass, resourceName);

        try {
            return objectMapper.readTree(json);
        } catch (IOException exception) {
            throw new UncheckedIOException("Cannot parse JSON test resource: " + resourceName, exception);
        }
    }
}
```

```java
import com.fasterxml.jackson.core.JsonProcessingException;
import com.fasterxml.jackson.databind.JsonNode;

import com.pivotService.test.JsonTestResources;
import org.assertj.core.api.AbstractAssert;

import java.time.Instant;
import java.time.format.DateTimeParseException;
import java.util.Arrays;
import java.util.LinkedHashSet;
import java.util.List;
import java.util.Set;

public final class JsonNodeAssert extends AbstractAssert<JsonNodeAssert, JsonNode> {

    private static final ObjectMapper PRETTY_OBJECT_MAPPER = new ObjectMapper();

    JsonNodeAssert(JsonNode actual) {
        super(actual, JsonNodeAssert.class);
    }

    public JsonNodeAssert hasOnlyTopLevelFields(String... expectedFields) {
        isNotNull();

        if (!actual.isObject()) {
            failWithMessage(
                    "Expected JSON root to be object, but was <%s>. Actual JSON:%n%s",
                    actual.getNodeType(),
                    pretty(actual)
            );
        }

        Set<String> actualFields = new LinkedHashSet<>();
        actual.fieldNames().forEachRemaining(actualFields::add);

        Set<String> expectedFieldSet = new LinkedHashSet<>(List.of(expectedFields));

        if (!actualFields.equals(expectedFieldSet)) {
            failWithMessage(
                    "Expected JSON top-level fields to be <%s>, but were <%s>. Actual JSON:%n%s",
                    expectedFieldSet,
                    actualFields,
                    pretty(actual)
            );
        }

        return this;
    }

    public JsonNodeAssert hasObjectAt(String pointer) {
        JsonNode node = requiredNodeAt(pointer);

        if (!node.isObject()) {
            failWithMessage(
                    "Expected JSON node at <%s> to be object, but was <%s>. Actual JSON:%n%s",
                    pointer,
                    node.getNodeType(),
                    pretty(actual)
            );
        }

        return this;
    }

    public JsonNodeAssert hasArrayAt(String pointer) {
        JsonNode node = requiredNodeAt(pointer);

        if (!node.isArray()) {
            failWithMessage(
                    "Expected JSON node at <%s> to be array, but was <%s>. Actual JSON:%n%s",

        }

    }

                    node.getNodeType(),
                    pretty(actual)
            );
        }

        if (node.size() != expectedSize) {
            failWithMessage(
                    "Expected JSON array at <%s> to have size <%s>, but had <%s>. Array:%n%s%nActual JSON:%n%s",
                    pointer,
                    expectedSize,
                    node.size(),
                    pretty(node),
                    pretty(actual)
            );
        }

        return this;
    }

    public JsonNodeAssert hasTextAt(String pointer, String expectedValue) {

    public JsonNodeAssert hasIsoInstantAt(String pointer) {

        } catch (DateTimeParseException exception) {
            failWithMessage(
                    "Expected JSON text at <%s> to be valid ISO-8601 Instant, but was <%s>. Actual JSON:%n%s",
                    pointer,
                    node.textValue(),
                    pretty(actual)
            );
        }

        return this;
    }

    public JsonNodeAssert doesNotHavePath(String pointer) {
        isNotNull();

        JsonNode node = actual.at(pointer);

        if (!node.isMissingNode()) {
            failWithMessage(
                    "Expected JSON path <%s> to be absent, but found:%n%s%nActual JSON:%n%s",
                    pointer,
                    pretty(node),
                    pretty(actual)
            );
        }

        return this;
    }

    public JsonNodeAssert isEqualToJsonResource(
            ObjectMapper objectMapper,
            Class<?> anchorClass,
            String resourceName
    ) {
        return isEqualToJsonResourceIgnoring(objectMapper, anchorClass, resourceName);
    }

    public JsonNodeAssert isEqualToJsonResourceIgnoring(
            ObjectMapper objectMapper,
            Class<?> anchorClass,
            String resourceName,
            String... ignoredPointers
    ) {
        isNotNull();

        JsonNode expected = JsonTestResources.readJson(objectMapper, anchorClass, resourceName);

        JsonNode normalizedExpected = copyWithoutIgnoredPointers(expected, ignoredPointers);
        JsonNode normalizedActual = copyWithoutIgnoredPointers(actual, ignoredPointers);

        if (!normalizedActual.equals(normalizedExpected)) {
            failWithMessage(
                    """
                    JSON contract mismatch for resource <%s>.
                    Ignored JSON pointers: %s
                    
                    Expected fixture:
                    %s
                    
                    Actual message:
                    %s
                    
                    Compared expected:
                    %s
                    
                    Compared actual:
                    %s
                    """,
                    resourceName,
                    Arrays.toString(ignoredPointers),
                    pretty(objectMapper, expected),
                    pretty(objectMapper, actual),
                    pretty(objectMapper, normalizedExpected),
                    pretty(objectMapper, normalizedActual)
            );
        }

        return this;
    }

    private JsonNode requiredNodeAt(String pointer) {
        isNotNull();

        JsonNode node = actual.at(pointer);

        if (node.isMissingNode()) {
            failWithMessage(
                    "Expected JSON path <%s> to exist, but it was missing. Actual JSON:%n%s",
                    pointer,
                    pretty(actual)
            );
        }

        return node;
    }

    private static JsonNode copyWithoutIgnoredPointers(JsonNode source, String... ignoredPointers) {
        JsonNode copy = source.deepCopy();

        Arrays.stream(ignoredPointers)
                .filter(pointer -> pointer != null && !pointer.isBlank())
                .forEach(pointer -> removeAt(copy, pointer));

        return copy;
    }

    private static void removeAt(JsonNode root, String pointer) {
        if (!pointer.startsWith("/")) {
            throw new IllegalArgumentException("JSON pointer must start with '/': " + pointer);
        }

        int lastSlashIndex = pointer.lastIndexOf('/');
        String parentPointer = lastSlashIndex == 0 ? "" : pointer.substring(0, lastSlashIndex);
        String leafToken = unescapeJsonPointerToken(pointer.substring(lastSlashIndex + 1));

        JsonNode parent = parentPointer.isEmpty() ? root : root.at(parentPointer);

        if (parent instanceof ObjectNode objectNode) {
            objectNode.remove(leafToken);
            return;
        }

        if (parent instanceof ArrayNode arrayNode) {
            arrayNode.remove(Integer.parseInt(leafToken));
        }
    }

    private static String unescapeJsonPointerToken(String token) {
        return token
                .replace("~1", "/")
                .replace("~0", "~");
    }

    private static String pretty(JsonNode jsonNode) {
        return pretty(PRETTY_OBJECT_MAPPER, jsonNode);
    }

    private static String pretty(ObjectMapper objectMapper, JsonNode jsonNode) {
        try {
            return objectMapper
                    .writerWithDefaultPrettyPrinter()
                    .writeValueAsString(jsonNode);
        } catch (JsonProcessingException exception) {
            return String.valueOf(jsonNode);
        }
    }
}
```

## Как тестировать последующие обновления данных

Для incremental update лучше не ждать scheduler через `Thread.sleep`. Хороший тест должен быть детерминированным.

Идея такая же:

```java
given(clickHouseService.executeAndExtract(anyString()))
        .willReturn(QueryDataFixtures.initialRowsForDashboard())
        .willReturn(QueryDataFixtures.changedRowsForDashboard());
```

Потом:

```java
sendJson(CONFIG_REQUEST_DESTINATION, "set-view-configuration.json");

JsonNode configurationUpdated = nextFrame("ConfigurationUpdated");
JsonNode initialData = nextFrame("initial ReceiveData");

assertConfigurationUpdated(configurationUpdated);
assertReceiveData(initialData, "receive-data.initial.expected.json");

// Здесь нужно вызвать твой реальный компонент, который инициирует отправку обновлений.
// Например:
// dashboardUpdateService.publishChangedRows(...);
// или
// tradeChangeListener.onTradesChanged(...);

JsonNode changedData = nextFrame("changed ReceiveData");

assertReceiveData(changedData, "receive-data.changed.expected.json");
```

Главное правило: **в тесте явно вызывай компонент, который публикует изменения**, а не жди, пока сработает настоящий scheduler.

## Коротко: что именно мокать

Для описанного кейса лучше мокать не `ResultSet`, не `JdbcTemplate`, а внешний boundary:

```java
@MockitoBean
private ClickHouseService clickHouseService;
```

И возвращать:

```java
given(clickHouseService.executeAndExtract(anyString()))
        .willReturn(QueryDataFixtures.initialRowsForDashboard());
```

То есть:

- `QueryData` — это test fixture/stub;
- `ClickHouseService` — Mockito mock;
- WebSocket/STOMP — реальные;
- JSON serialization contract — реальный;
- DB/Testcontainers — не нужны.
